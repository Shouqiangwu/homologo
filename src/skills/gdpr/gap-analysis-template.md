# GDPR Gap Analysis 输出模板

> 适用于车联网场景下的GDPR合规差距分析

---

## 检查项清单

### A. 合法性基础

| 编号 | 检查项 | 法规依据 |
|------|--------|---------|
| GDPR-A01 | 每项数据处理是否有明确的Art.6合法性基础 | Art.6 |
| GDPR-A02 | 同意是否与购车/服务合同分别获取 | EDPB GL §56-58 |
| GDPR-A03 | 拒绝同意是否不影响车辆基本功能 | EDPB GL §54 |
| GDPR-A04 | Art.9特殊类别数据是否有明确同意 | Art.9(2)(a) |
| GDPR-A05 | ePrivacy Art.5(3)车端数据存取同意 | ePrivacy Art.5(3) |

### B. 数据分类和处理

| 编号 | 检查项 | 法规依据 |
|------|--------|---------|
| GDPR-B01 | 是否建立了车辆数据分类清单 | Art.30 |
| GDPR-B02 | DMS数据是否识别为生物识别数据 | Art.9 |
| GDPR-B03 | 位置数据是否有特殊保护措施 | EDPB GL §36-38 |
| GDPR-B04 | VIN相关数据是否按个人数据处理 | EDPB GL §28-32 |
| GDPR-B05 | 车端本地处理是否优先于云端处理 | EDPB GL §69-72 |
| GDPR-B06 | DMS/ADAS原始图像是否车端处理后立即删除 | GSR Art.7(3) + GDPR |

### C. 数据主体权利

| 编号 | 检查项 | 法规依据 |
|------|--------|---------|
| GDPR-C01 | 分层隐私告知（书面+车载+APP） | Art.13-14 |
| GDPR-C02 | 车载"一键删除"功能 | Art.17 + EDPB GL |
| GDPR-C03 | 数据访问和可携权功能 | Art.15, 20 |
| GDPR-C04 | 车辆转售时数据清除流程 | Art.17 |
| GDPR-C05 | 多驾驶员档案管理系统 | EDPB GL §87-89 |

### D. 跨境数据传输

| 编号 | 检查项 | 法规依据 |
|------|--------|---------|
| GDPR-D01 | EU→中国传输流已识别 | Art.44 |
| GDPR-D02 | EU SCCs已签署 | Art.46(2)(c) |
| GDPR-D03 | TIA（传输影响评估）已完成 | Schrems II判决要求 |
| GDPR-D04 | 补充措施已实施 | EDPB关于补充措施的建议 |
| GDPR-D05 | 同意未被用作系统性传输基础 | Art.49(1)(a) |
| GDPR-D06 | PIPL跨境传输合规（反向） | PIPL Art.38-43 |

### E. 安全和组织措施

| 编号 | 检查项 | 法规依据 |
|------|--------|---------|
| GDPR-E01 | DPIA已完成（大规模位置追踪/DMS） | Art.35 |
| GDPR-E02 | DPO或EU代表已任命 | Art.37 / Art.27 |
| GDPR-E03 | 数据泄露通知流程（72小时） | Art.33-34 |
| GDPR-E04 | 处理记录（ROPA） | Art.30 |
| GDPR-E05 | 与R155 CSMS的数据保护整合 | R155 + GDPR Art.32 |

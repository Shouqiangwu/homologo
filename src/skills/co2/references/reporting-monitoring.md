# 年度监控和报告流程

> EU 2021/392 实施法规

---

## 报告流程

1. **成员国收集数据**：各成员国收集其境内注册的所有新车CO2排放数据
2. **报送EEA**：成员国在次年2月28日前将数据报送欧洲环境署（EEA）
3. **EEA汇总发布**：EEA发布初步数据，制造商可核查
4. **欧委会计算**：欧委会计算每个制造商的车队平均排放和特定目标
5. **超标通知**：如超标，通知制造商缴纳罚款

## 报告数据要素

每辆新注册车辆需报告：VIN、CO2排放值（WLTP）、整备质量、轮距、足印面积、燃料类型、电续航里程（如适用）等。

## 交叉引用

- **→ isc/**：实际排放（real-world）监控与型式认证排放的一致性

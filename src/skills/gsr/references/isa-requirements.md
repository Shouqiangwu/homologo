# ISA 智能限速辅助系统要求

> EU 2021/1958 — 补充EU 2019/2144 Art.6

---

## 适用范围

- **适用类别：** 全部M类和N类机动车
- **新车型强制日期：** 2022-07-06
- **所有新车强制日期：** 2024-07-07
- **状态：** ✅ 已完全强制

## 系统定义

ISA是一个**辅助系统**（非控制系统），通过以下方式提示和鼓励驾驶员在超速时减速：
- 加速踏板反馈（触觉）
- 声觉警告（级联式）
- 视觉警告
- 或以上组合

驾驶员始终可以通过持续踩油门覆盖ISA。ISA**不得自动制动**来降低车速。

## 核心技术要求

### 限速识别 — EU 2021/1958 Art.3-4

| 要求项 | 具体阈值/要求 | 法规条款 |
|--------|-------------|---------|
| 识别信息源 | 交通标志识别（摄像头）和/或地图数据，或两者融合 | Art.3(1) |
| 识别准确率 | **≥90%**（在所有EU成员国内） | Art.4(2) |
| 识别标志类型 | 必须覆盖EU/EEA成员国使用的限速标志格式 | Art.4(1) |
| 隐含限速 | 应能识别进入城区/离开城区等隐含限速变化 | Art.4(1) |
| 临时限速 | 应能识别施工区等临时限速标志 | Art.4(1) |
| 速度限制单位 | km/h（全EU统一） | — |

### 驾驶员反馈方式 — EU 2021/1958 Art.5

车企可选择以下4种反馈方式之一或组合：

| 反馈方式 | 说明 | 要求 |
|---------|------|------|
| **方式A：级联声觉警告** | 超速后发出递进式声觉警告 | 超速后≤X秒开始，持续至驾驶员减速或确认 |
| **方式B：振动反馈** | 通过加速踏板振动提示 | 超速时踏板振动 |
| **方式C：加速踏板回推力** | 增加油门踏板阻力 | 驾驶员可通过施加更大力量覆盖 |
| **方式D：速度控制功能** | 自动限制发动机输出（非制动） | 驾驶员可通过kick-down覆盖 |

**关键要求：** 无论选择哪种方式，驾驶员都必须能随时覆盖（override）ISA。

### 系统状态管理 — EU 2021/1958 Art.6

| 要求项 | 具体要求 |
|--------|---------|
| 默认状态 | 每次启动车辆后ISA必须处于**激活状态** |
| 可关闭性 | 驾驶员可手动关闭ISA，但每次启动后自动恢复 |
| 关闭提醒 | 系统关闭时必须向驾驶员显示明确提示 |
| 速度显示 | 必须向驾驶员显示当前识别的限速值 |

### 数据记录限制 — EU 2021/1958 Art.7 + EU 2019/2144 Art.6(5)

| 限制项 | 具体要求 |
|--------|---------|
| 不得持续记录 | ISA不得持续记录或保留位置数据和驾驶行为数据 |
| 闭环处理 | 数据仅在系统运行所需范围内处理，处理后立即删除 |
| 第三方访问 | 数据不得在任何时候被第三方访问或提供给第三方 |
| **→ 交叉引用：** | 见 gdpr/ 领域的车辆数据合规要求 |

## 合规检查清单

- [ ] ISA系统是否安装在车辆上并默认激活？
- [ ] 限速识别信息源是什么？（摄像头/地图/融合）
- [ ] 限速识别准确率是否≥90%？（需提供测试报告）
- [ ] 是否覆盖所有EU/EEA成员国的限速标志格式？
- [ ] 是否能识别隐含限速（城区入口/出口等）？
- [ ] 是否能识别临时限速标志（施工区等）？
- [ ] 驾驶员反馈方式是哪种（A/B/C/D）？是否符合对应技术要求？
- [ ] 驾驶员是否可以随时覆盖ISA？
- [ ] 每次启动后ISA是否自动恢复激活状态？
- [ ] 系统关闭时是否有明确提示？
- [ ] 是否向驾驶员显示当前识别的限速值？
- [ ] ISA是否**不**持续记录位置数据或驾驶行为数据？
- [ ] 数据处理后是否立即删除？
- [ ] 数据是否**不**可被第三方访问？

## 常见不合规情况

1. **限速标志数据库仅覆盖中国/亚洲标志**：中国GB5768标准与欧洲维也纳公约体系的标志设计差异显著。需要为EU27+EEA国家逐国适配标志识别模型。这是中国车企最常见也最耗时的适配工作。

2. **地图数据供应商不覆盖EU**：部分中国车企使用百度/高德地图数据，需切换至HERE/TomTom等覆盖EU的供应商。

3. **准确率测试方法不符**：EU 2021/1958 Annex中定义了按国家分组的限速标志目录（exhaustive catalogue），准确率必须基于此目录测试，而非车企自定义的测试集。

4. **数据隐私不合规**：ISA记录了GPS轨迹或驾驶行为数据用于系统改进/远程诊断，违反Art.7的数据最小化要求。

5. **kick-down覆盖逻辑问题**：方式D（速度控制）的覆盖机制必须通过明确的驾驶员意图动作（如kick-down）触发，不得仅靠轻踩油门就自动解除。

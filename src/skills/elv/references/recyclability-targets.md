# 可回收率与可利用率目标

> ELV Directive 2000/53/EC Art.7 + EU 2005/64/EC（型式认证） + ISO 22628

---

## 目标值和适用日期

| 指标 | 目标 | 计算方法 | 适用日期 |
|------|------|---------|---------|
| **可重用性+可回收性**（Reusability + Recyclability Rate, Rcyc） | ≥85%（整车质量） | ISO 22628 | 2015-01-01起新车型 |
| **可重用性+可回收性+可回收能量**（Reusability + Recoverability Rate, Rcov） | ≥95%（整车质量） | ISO 22628 | 2015-01-01起新车型 |

**监管层级：** 
- ELV Directive设定最终目标
- EU 2005/64/EC（已整合入EU 2018/858）规定型式认证阶段必须提交证明
- 2018起改造提案要求OEM额外提供"材料使用计划"

## ISO 22628 计算方法

### 质量分类体系

```
整车质量 (m_v)
├─ 可重用（Reusable）部件质量 (m_P)
│   └─ 可直接再使用、无需加工
├─ 可回收（Recyclable）材料质量 (m_M)
│   └─ 可通过材料回收得到新原料
├─ 可回收能量（Energy Recovery）质量 (m_TE)
│   └─ 可通过焚烧回收能量
└─ 最终处置（Disposal）质量 (m_d)
    └─ 填埋
```

### 计算公式

```
Rcyc = (m_P + m_D + m_M) / m_v × 100%    ≥85%
Rcov = (m_P + m_D + m_M + m_Te) / m_v × 100%    ≥95%

其中：
m_P = 预处理阶段可回收材料（油液、轮胎、电池等）
m_D = 拆解阶段可回收/重用材料
m_M = 切碎后金属和非金属材料回收
m_Te = 能量回收
```

## 五步计算流程（ISO 22628 §6）

```
Step 1: 预处理阶段（Pre-treatment）
├─ 油液排放（燃油、冷却液、制动液、空调制冷剂...）
├─ 拆除轮胎
├─ 拆除蓄电池
├─ 拆除催化转化器
└─ 安全气囊中和
   → 计算m_P（所有预处理移除材料按可回收性归类）

Step 2: 拆解阶段（Dismantling）
├─ 拆除主要零部件（车门、座椅、仪表板、保险杠...）
└─ 可按翻新重用归类
   → 计算m_D

Step 3: 金属分离（Metal separation）
└─ 车身残骸破碎后金属提取
   → 计算m_M金属部分

Step 4: 非金属残渣处理（Non-metallic residue, "ASR"）
├─ 先进技术分选（塑料分选）
├─ 能量回收（焚烧）
└─ 填埋部分
   → 计算m_M非金属部分、m_Te、m_d

Step 5: 汇总计算Rcyc和Rcov
```

## 典型质量分布参考值

| 系统/部件 | 整车占比 | 可回收性分类 |
|----------|---------|------------|
| 车身（钢/铝） | 25-35% | 高（>95%作为m_M） |
| 底盘 | 15-20% | 高（钢作为m_M） |
| 动力总成（ICE） | 15-20% | 中-高（翻新重用+金属回收） |
| BEV电池 | 25-30%（BEV） | 中（2025前目标50%，2030 75%） |
| 内饰（塑料/织物） | 10-15% | 低（塑料分选率低，部分m_Te） |
| 玻璃 | 3-5% | 中（可回收但污染常见） |
| 轮胎 | 3-4% | 中（橡胶粉/能量回收） |
| 油液 | 1-2% | 高（大部分可回收/处理） |

## BEV vs ICE的可回收率差异

### BEV挑战

| 问题 | 影响 |
|------|------|
| 电池质量大 | 整车25-30%质量集中在电池 |
| 电池回收率计入方式 | 必须按Battery Reg Art.6(23)和新ELV协调 |
| 高压电驱动 | 稀土磁体回收困难 |
| 热管理复杂 | 电池冷却液污染较高 |

### ICE优势

| 项目 | 原因 |
|------|------|
| 成熟回收链 | 40+年积累的钢铁/铝回收工艺 |
| 动力总成翻新 | 发动机/变速箱可整体再制造 |
| 成分简单 | 钢铁占比高，回收效率高 |

## 型式认证中的提交要求

### 必交文件（EU 2018/858 Annex IV Part I 项目51/57）

| 文件 | 内容 |
|------|------|
| ISO 22628计算报告 | 各系统质量分布、分类、Rcyc/Rcov计算 |
| 材料清单 | 所有材料按类别（金属、塑料、玻璃等）的质量 |
| 零部件清单 | 主要零部件的材料组成 |
| 可回收性声明 | OEM承诺达到Rcyc≥85%, Rcov≥95% |
| 拆解信息（IDIS） | 预处理和拆解步骤 |
| 再生材料使用声明（新ELV提案后） | 回收塑料/钢/铝的使用比例 |

### Technical Service审核

- 抽查材料清单中质量占比前100个零部件
- 现场查阅IMDS数据
- 核对拆解手册与计算一致性

## 回收材料含量（新ELV提案）

现行ELV无强制要求，但新ELV法规提案（EU 2023/0284）将引入：

| 材料 | 最低回收含量（新车中） | 适用日期 |
|------|---------------------|---------|
| 塑料 | 25%（6年后可能升至30%） | 法规通过后6年 |
| 钢 | 拟讨论中，可能10-15% | 法规通过后6年 |
| 铝 | 拟讨论中 | - |

## 合规检查清单

- [ ] 是否按ISO 22628完成了Rcyc/Rcov计算？
- [ ] Rcyc≥85%且Rcov≥95%？
- [ ] 计算报告是否纳入型式认证提交？
- [ ] 整车质量分布数据是否覆盖所有系统？
- [ ] BEV电池质量是否正确计入（按Battery Reg协调）？
- [ ] 内饰塑料的回收路径是否明确（材料回收vs能量回收）？
- [ ] 新材料（碳纤维、复合材料）是否对回收率造成负面影响？
- [ ] 是否跟踪了新ELV提案的再生材料含量要求？
- [ ] 供应商是否提供了材料组成数据（IMDS）？

## 常见不合规情况

1. **复合材料占比过高**：采用大量碳纤维增强塑料（CFRP）降低Rcyc（CFRP回收率低，常只能m_Te）。
2. **粘胶连接过多**：车身/电池包大量使用结构胶，拆解难度大导致m_D降低。
3. **电池包一体化设计**：CTB/CTC（Cell-to-Body/Chassis）电池使Rcyc计算复杂化。
4. **内饰混合材料**：多层复合内饰（皮革+泡沫+塑料+织物）分选困难，大部分只能m_Te。
5. **BEV电池质量未正确归类**：Battery Reg要求2030年回收率75%，但型式认证时按旧假设计算。
6. **供应商材料数据缺失**：IMDS数据不全，只能用保守估计，导致Rcyc偏低。

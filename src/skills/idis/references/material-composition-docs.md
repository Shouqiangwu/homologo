# 材料成分文档

> ISO 1043 / ISO 11469 / IMDS / IDIS材料模块

---

## 材料文档的作用

IDIS拆解信息中的材料成分文档用途：

1. **指导拆解**：拆解技师知道哪些部件是什么材料，决定回收路径
2. **有害物质追踪**：定位Annex II豁免物质
3. **Rcyc/Rcov计算**：为ISO 22628提供质量分布
4. **新ELV提案准备**：为回收材料含量验证提供基础

## 材料标识标准

### ISO 1043-1（塑料基本符号）

| 符号 | 材料 | 汽车应用 |
|------|------|---------|
| PP | 聚丙烯 | 保险杠、仪表板、内饰 |
| PE | 聚乙烯 | 燃油箱、容器 |
| PVC | 聚氯乙烯 | 线束护套、密封条 |
| PUR | 聚氨酯 | 座椅泡沫、方向盘 |
| ABS | 丙烯腈-丁二烯-苯乙烯 | 内饰硬件 |
| PC | 聚碳酸酯 | 灯罩、全景天窗 |
| PA | 聚酰胺（尼龙） | 发动机罩、进气歧管 |
| POM | 聚甲醛 | 齿轮、轴承 |
| PBT | 聚对苯二甲酸丁二醇酯 | 电子连接器 |
| EPDM | 乙丙橡胶 | 密封条 |
| NBR | 丁腈橡胶 | 油封 |

### ISO 1043-2（填充和增强符号）

| 符号 | 含义 |
|------|------|
| -T | 滑石（Talc） |
| -G | 玻璃（Glass） |
| -GF | 玻璃纤维 |
| -CF | 碳纤维 |
| -MD | 矿物粉（Mineral） |
| 数字 | 含量百分比 |

**示例：** `PP-T20`表示含20%滑石的聚丙烯；`PA6-GF30`表示含30%玻璃纤维的PA6。

### ISO 11469（塑料件标识）

要求塑料件上永久标识材料类型：

| 要求 | 说明 |
|------|------|
| 位置 | 易观察到的位置 |
| 字符高度 | ≥3 mm（一般），≥1 mm（小件允许） |
| 标识方法 | 模塑、冲压、激光、永久标签 |
| 内容 | `> PP-T20 <`（含括号符号） |
| 适用范围 | 质量≥100 g的塑料件（推荐）|

### ISO 15270（塑料回收标识）

回收级别标识：

| 级别 | 含义 |
|------|------|
| 1 | 热塑性原材料可无限回收 |
| 2 | 热塑性可降级回收 |
| 3 | 热固性不可材料回收（仅能量） |

## IDIS中的材料数据

### 数据粒度

IDIS材料信息的典型粒度：

| 级别 | 覆盖 |
|------|------|
| Vehicle level | 整车材料汇总（最粗） |
| System level | 主要系统（动力、底盘、车身、内饰）|
| Module level | 模组（保险杠总成、仪表板总成）|
| Component level | 零部件（保险杠面板、泡沫块）|
| Material level | 均质材料（最细） |

**IDIS要求：** 至少Module level，推荐Component level。

### 数据字段

每个零部件的材料数据应包括：

| 字段 | 示例 |
|------|------|
| Part number | BYD-PN-12345 |
| Part name | 前保险杠面板 |
| Quantity per vehicle | 1 |
| Total weight (g) | 3,200 |
| Main material | PP-T20 |
| Material content % | 100% |
| Sub-materials | 颜料<0.5%、添加剂<1% |
| ISO 11469 marking | 是/否 + 位置 |
| SVHC (>0.1%) | 是/否 + 物质名 |
| Annex II exemption | 豁免条目号 |

## IMDS与IDIS的关系

### IMDS（International Material Data System）

汽车行业材料数据申报的事实标准：

| 项目 | 详情 |
|------|------|
| 运营方 | EDS（西门子子公司） |
| 用户 | 全球OEM + 供应商 |
| 数据粒度 | 均质材料（最细） |
| 覆盖 | 全球销售的车型 |
| 强制性 | OEM要求供应商提交，非法规直接强制 |

### IMDS vs IDIS

| 维度 | IMDS | IDIS |
|------|------|------|
| 目的 | 供应链材料数据 | 拆解信息 |
| 用户 | OEM和供应商 | ATF和拆解企业 |
| 粒度 | 最细（均质材料） | 中等（零部件级） |
| 关注 | 合规（REACH/ELV/Battery） | 实操（如何拆解） |
| 数据格式 | IMDS专有 | IDIS标准 |

### 数据流

```
供应商 → IMDS（材料明细）
    ↓
OEM汇总 → IDIS（拆解导向的零部件材料）
    ↓
ATF使用 → 指导拆解和回收
```

## 特殊材料的标识

### 有害物质（Annex II豁免）

| 物质 | IDIS标识方式 |
|------|------------|
| 含铅焊料 | "Pb"标签位置+豁免条目号 |
| 铅合金钢 | "Pb"位置+2(c)(i) |
| 铅酸电池 | "Pb"+电池标签 |
| 六价铬（如残留） | "Cr(VI)"+警告 |

### 稀土（回收价值）

| 应用 | 含稀土 |
|------|-------|
| 驱动电机磁铁 | Nd, Dy（高价值） |
| 扬声器磁铁 | Nd |
| 电动助力转向磁铁 | Nd |
| LED照明 | 少量 |

IDIS应指明稀土磁铁位置和回收价值。

### 新兴材料

| 材料 | 挑战 |
|------|------|
| 碳纤维复合材料（CFRP） | 回收技术不成熟 |
| 铝合金泡沫 | 分选困难 |
| 生物基塑料 | 与传统塑料分选 |
| 形状记忆合金（SMA） | 特殊金属 |
| 石墨烯涂层 | 追踪困难 |

## 与新ELV提案的关系

新ELV提案（EU 2023/0284）强化材料数据要求：

| 新要求 | 影响 |
|--------|------|
| 材料护照 | 比IDIS更详细的数字化记录 |
| 回收含量追踪 | 需证明塑料/钢/铝中PCR比例 |
| 关键原材料（CRM）追踪 | 稀土、锂、钴等 |
| 互操作（Battery Passport） | 与电池数字护照对接 |

## 合规检查清单

- [ ] 所有≥100g塑料件是否有ISO 11469标识？
- [ ] 标识位置是否在IDIS中指出？
- [ ] 材料数据是否至少覆盖Module级别？
- [ ] 供应商IMDS数据是否完整（覆盖≥90%质量）？
- [ ] BEV电池材料成分是否详细？
- [ ] 稀土磁铁位置是否标注？
- [ ] Annex II豁免物质位置是否清楚？
- [ ] SVHC >0.1%的零部件是否标注？
- [ ] 碳纤维/复合材料是否特别标识？
- [ ] 材料数据是否与新ELV提案兼容？

## 常见不合规情况

1. **ISO 11469标识缺失**：量产后才发现多数塑料件无标识。
2. **IDIS与IMDS数据脱节**：IMDS有数据但IDIS未引用。
3. **电池材料简化**：仅说"Li-ion"，未详细化学类型（LFP vs NMC）。
4. **稀土位置遗漏**：驱动电机/扬声器磁铁位置未标注。
5. **CFRP未特殊标识**：与普通塑料混在一起，回收时污染塑料流。
6. **供应链数据不全**：小供应商未提交IMDS数据。
7. **新材料未登记**：新工艺（如聚醚醚酮PEEK）在IDIS中无记录。
8. **Annex II豁免号不准确**：引用老版Annex II的豁免号。

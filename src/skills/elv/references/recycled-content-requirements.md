# 再生材料含量要求（新ELV提案）

> EU 2023/0284(COD) Art.6 — 立法进行中

---

## 核心目标

新ELV法规提案强制新车中使用一定比例的**回收/再生材料**（Recycled Content），推动汽车行业从线性经济向循环经济转型。现行ELV指令无此要求——这是提案的最重要创新。

## 提案数值（当前文本2024-06理事会版）

### 塑料回收含量

| 节点 | 新车塑料总量中回收料比例 | 其中来自ELV的比例 |
|------|------------------------|-----------------|
| **生效后6年** | 25% | 至少25%（即总量6.25%） |
| **生效后10年** | 30% | 至少30%（即总量9%） |
| 议会进一步提议 | 至2035年40% | - |

### 钢和铝（谈判中）

| 材料 | 议会提议 | 理事会立场 | 最终可能 |
|------|---------|----------|---------|
| 钢 | 20% | 待定 | 15-20% |
| 铝 | 10% | 待定 | 5-10% |
| 生效时间 | 法规生效后6年 | - | 2030-2032 |

### 稀土和关键原材料（Critical Raw Materials, CRMs）

| 材料 | 讨论中 |
|------|--------|
| 永磁体中稀土（Nd, Dy等） | 可能最低20%回收含量 |
| 锂、钴、镍 | 与Battery Reg协调（Battery Reg已有要求） |
| 铜 | 可能未来加入 |

## "回收含量"的定义

### 三类回收料

| 类型 | 定义 | 计入方式 |
|------|------|---------|
| **Pre-consumer（消费前）** | 生产过程中的废料/边角料 | 不计入（视为原生料副产品） |
| **Post-consumer（消费后）** | 终端产品报废后回收的材料 | 完全计入 |
| **Post-industrial（工业后）** | 其他行业产品生产边角料 | 部分计入（视源头） |

### "来自ELV"的特殊要求

塑料回收要求中"至少25%来自ELV"意味着：
- 不能完全用包装、家电等其他来源的回收塑料填充
- 必须有从报废汽车中回收的塑料
- 推动"闭环回收"（closed-loop recycling）

**挑战：** 目前EU从ELV中回收的塑料品质有限（多为PP、HDPE），高性能塑料（PC/ABS、PA6/PA66）回收链不成熟。

## 验证与报告

### 质量平衡法（Mass Balance）

OEM可采用EN 45557或类似标准的质量平衡方法证明回收含量：

```
材料入厂时按回收料vs原生料比例混合
     ↓
整个生产流程按比例分配回收料"虚拟配额"
     ↓
出厂成品按配额声明回收含量
```

### 第三方认证

| 认证 | 说明 |
|------|------|
| **ISCC PLUS** | 化工行业常用回收认证 |
| **RecyClass** | 塑料回收专用 |
| **EuCertPlast** | 欧盟塑料回收认证 |
| **TÜV Rheinland** | 第三方验证 |

### OEM报告义务

| 频次 | 内容 |
|------|------|
| 每年 | 上年度各车型回收含量实际值 |
| 新车型 | 型式认证时声明目标回收含量 |
| 供应链 | 保存5年的供应商声明和第三方认证记录 |

## 不同塑料品类的回收难度

| 塑料类型 | 车辆应用 | 回收可行性 | PCR供应链成熟度 |
|---------|---------|----------|---------------|
| PP（聚丙烯） | 保险杠、仪表板、杂物箱 | 高 | 成熟 |
| HDPE/LDPE | 燃油箱、管路 | 高 | 成熟 |
| PC/ABS | 内饰件、格栅 | 中 | 发展中 |
| PA6/PA66 | 发动机罩、进气歧管 | 中-低 | 弱 |
| PUR（聚氨酯） | 座椅泡沫、方向盘 | 低 | 极弱 |
| PVC | 线束护套、密封条 | 低（含卤素） | 较弱 |
| 纤维增强复合材料 | 车身、结构件 | 极低 | 无 |
| 热塑性弹性体（TPO） | 保险杠、内饰 | 中 | 发展中 |

**合规策略：** 
- 优先在PP/HDPE件上使用PCR，保障基础合规
- 通过减少PUR和复合材料使用，降低母子供应链难度
- 与专业回收商（如Suez, Veolia, Remondis）建立战略合作

## 成本影响

### 原生料 vs PCR成本对比（2024年欧洲市场）

| 材料 | 原生料价格（EUR/kg） | PCR价格 | 溢价 |
|------|--------------------|--------|------|
| PP | 1.2-1.5 | 1.5-2.0 | +25-35% |
| HDPE | 1.3-1.6 | 1.6-2.1 | +25-30% |
| PC/ABS | 3.0-3.5 | 4.0-5.0 | +30-40% |
| PA6 | 3.5-4.5 | 5.0-6.5 | +40-45% |

### 单车成本影响估算

假设车辆塑料总量150 kg，25%来自PCR，平均溢价30%：
```
增量成本 = 150 × 0.25 × (原生料均价1.8 EUR/kg) × 0.3 = 约€20/辆
长期（30% PCR）：约€27/辆
```

加上质量平衡系统、第三方认证、供应链管理成本，实际单车增量可能达**€40-80**。

## 对中国车企的战略选择

### 选项1：在EU本地采购PCR

| 优势 | 劣势 |
|------|------|
| 合规直接 | 成本较高 |
| 建立在EU供应链 | 对中国基地无价值 |
| 物流简化 | 依赖EU回收商产能 |

### 选项2：中国基地建立PCR产线，出口EU

| 优势 | 劣势 |
|------|------|
| 成本较低 | 海运增加 |
| 支撑国内市场未来需求 | 需证明PCR来自EU ELV（增加监管难度） |
| 垂直整合 | 初期投入大 |

### 选项3：混合策略

- EU产能使用EU PCR
- 中国产能建立符合ISCC PLUS的认证体系
- 高价值件（PA/PC-ABS）优先EU PCR

## 常见风险与误区

1. **只关注数值忽视"来自ELV"**：用食品包装PCR冲量，不符合"闭环"要求。
2. **低估PA/PC-ABS的PCR短缺**：发动机罩、格栅等高性能件找不到足够PCR。
3. **供应链认证不彻底**：中间商宣称PCR但无第三方认证支持。
4. **性能测试失败**：PCR材料加入后机械性能下降，车型需重新OEM内部测试（追加成本）。
5. **时间窗口误判**：认为"6年过渡期"很长，实际需要≥3年建立供应链。
6. **依赖Mass Balance全覆盖**：过度依赖虚拟配额，未做物理回收投资。

# 电池护照要求

> EU 2023/1542 Art.77 + Annex XIII

---

## 适用范围和时间

- **适用电池类别：** EV电池 + 工业可充电电池(>2kWh) + LMT电池
- **强制日期：** 2027-02-18
- **SLI电池和便携式电池：** 不适用

## 电池护照定义

每块适用电池必须配备一个**唯一的数字记录**，通过QR码可访问，包含静态数据（制造信息）和动态数据（使用和健康状态），按访问权限对不同用户群体开放。

## 数据要求 — Annex XIII

### 公共可访问数据

| 数据类别 | 具体参数 |
|---------|---------|
| 电池标识 | 唯一标识符、制造商名称、制造工厂、制造日期、型号、批次号 |
| 化学组成 | 正极/负极/电解液化学体系（如NCM811、LFP等）、有害物质含量 |
| 碳足迹 | 碳足迹声明值（kg CO₂e/kWh）、性能等级 |
| 回收材料含量 | 钴、铅、锂、镍的回收材料百分比 |
| 性能参数 | 额定容量(Ah)、额定能量(kWh)、额定电压(V)、标称循环寿命 |
| 标签信息 | CE标志状态、分类收集标志 |

### 授权用户可访问数据（监管机构、维修商、回收商）

| 数据类别 | 具体参数 |
|---------|---------|
| 详细技术文档 | BMS参数、安全测试报告 |
| 供应链信息 | 原材料来源国、尽职调查报告摘要 |
| 拆卸和回收指南 | 安全拆卸步骤、材料标识、危险信息 |
| 健康状态（SoH） | 当前容量/初始容量比、内阻变化、循环次数 |

### 动态数据（使用过程中更新）

| 数据 | 说明 |
|------|------|
| SoH（健康状态） | 通过BMS读取，定期更新至电池护照 |
| 维修/维护记录 | 更换模块、软件更新等 |
| 所有权变更 | 电池所有者信息（注意GDPR要求） |

## 技术实现要求

| 要求项 | 具体要求 |
|--------|---------|
| 唯一标识 | 每块电池一个唯一数字标识符 |
| QR码 | 物理标签上必须有QR码，链接至数字护照 |
| 数据格式 | 开放标准、可互操作格式 |
| 数据平台 | 必须使用符合EU要求的数字基础设施（去中心化系统） |
| 数据保留 | 电池全生命周期+报废后保留一定年限 |
| 数据安全 | 符合GDPR和网络安全要求 |

## 合规检查清单

- [ ] 电池是否配备了唯一数字标识符？
- [ ] 是否有物理QR码标签，可链接至数字护照？
- [ ] 公共可访问数据是否完整（标识、化学组成、碳足迹、回收含量、性能参数）？
- [ ] 授权用户数据是否可供监管机构/维修商/回收商访问？
- [ ] BMS是否能输出SoH动态数据至电池护照系统？
- [ ] 数据格式是否为开放标准、可互操作？
- [ ] 数据平台是否符合EU数字基础设施要求？
- [ ] 数据保护措施是否符合GDPR？

## 常见不合规情况

1. **无数字护照系统**：中国车企尚未建立EU合规的电池护照数字基础设施。
2. **数据平台封闭**：使用私有云平台，不满足可互操作性要求。
3. **SoH数据不开放**：BMS健康状态数据被OEM锁定，第三方维修商无法访问。
4. **QR码缺失**：电池包上无物理QR码标签。
5. **数据保留不足**：电池报废后数据被删除。

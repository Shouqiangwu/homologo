# ISO 22628 可回收率/可利用率计算

> ISO 22628:2002 + EU 2005/64/EC + IDIS平台实施

---

## ISO 22628方法论

ISO 22628:2002《道路车辆——可回收性和可再使用性——计算方法》定义了汽车可回收率（Recyclability Rate, Rcyc）和可利用率（Recoverability Rate, Rcov）的计算方法。

### 目标值（ELV Directive Art.7）

| 指标 | 符号 | 目标 | 生效日 |
|------|------|------|--------|
| 可回收率 | Rcyc | ≥85% | 2015起新车型 |
| 可利用率 | Rcov | ≥95% | 2015起新车型 |

## 五步计算程序（ISO 22628 §6）

### Step 1：预处理阶段（Pre-treatment）

计算ELV预处理阶段移除材料的质量 `mP`：

| 移除项 | 典型质量（M1乘用车） | 归类 |
|--------|-------------------|------|
| 燃油 | 40-60 kg（ICE）| 能量回收或处置 |
| 机油和油液 | 10-15 kg | 部分可回收 |
| 空调制冷剂 | 0.5-0.8 kg | 销毁/再生 |
| 轮胎 | 30-40 kg | 橡胶粉/能量回收 |
| 启动电池 | 12-20 kg | 铅酸回收 |
| 催化转化器 | 3-5 kg | 贵金属回收 |
| 大型塑料件（保险杠等） | 视拆解 | 材料回收 |
| EV电池（如有） | 250-500 kg | 单独计算，见 battery/ |

**计算：** 将 `mP` 按最终处理方式分配到可重用(m_Pr)、可回收材料(m_Pm)、能量回收(m_Pe)、处置(m_Pd)。

### Step 2：拆解阶段（Dismantling）

计算拆解阶段可回收/重用零部件的质量 `mD`：

| 典型零部件 | 重用潜力 | 材料回收 |
|----------|---------|---------|
| 发动机（ICE） | 高（再制造） | 金属 |
| 变速箱 | 高（再制造） | 金属 |
| 座椅总成 | 中 | 泡沫+金属 |
| 车门 | 中 | 钢+铝+玻璃 |
| 后视镜 | 高 | - |
| 保险杠 | 低 | 塑料回收 |
| 玻璃（挡风/车窗） | 低 | 夹层玻璃回收 |

### Step 3：金属分离（Metal Separation）

破碎后的金属提取：

| 金属 | 回收率（典型） |
|------|-------------|
| 钢 | 95-98% |
| 铝 | 90-95% |
| 铜（电线） | 85-90% |
| 贵金属（Au/Ag/Pt） | 依处理技术 |

**计算：** `mM` 金属部分 = 破碎金属质量 × 回收率因子

### Step 4：非金属残渣（ASR, Automotive Shredder Residue）

剩余的非金属残渣处理：

| 处理路径 | 典型占比 |
|---------|---------|
| 先进塑料分选 | 10-20% |
| 能量回收（焚烧） | 40-60% |
| 填埋 | 20-40% |

**计算：** `mM` 非金属部分 + `mTe` + `md` = ASR总质量

### Step 5：汇总

```
Rcyc = (mPr + mPm + mD重用和材料回收 + mM) / mv × 100%
Rcov = Rcyc + (mPe + mTe) / mv × 100%

其中 mv = 整车整备质量（kg）
```

## 计算示例（M1 BEV SUV）

### 车辆参数

- 整备质量 mv = 1,800 kg
- BEV电池 = 500 kg（占27.8%）

### 各阶段质量分布

| 阶段 | 材料分类 | 质量(kg) | 占比 |
|------|---------|---------|------|
| 预处理 | 油液+制冷剂 | 15 | 0.8% |
| 预处理 | 轮胎 | 35 | 1.9% |
| 预处理 | 12V电池 | 12 | 0.7% |
| 预处理 | **EV电池** | **500** | **27.8%** |
| 拆解 | 动力电机+减速器 | 80 | 4.4% |
| 拆解 | 座椅/内饰 | 150 | 8.3% |
| 拆解 | 主要塑料件 | 100 | 5.6% |
| 拆解 | 玻璃 | 40 | 2.2% |
| 破碎 | 车身钢 | 600 | 33.3% |
| 破碎 | 车身铝 | 150 | 8.3% |
| 破碎 | 铜（电线） | 30 | 1.7% |
| ASR | 塑料/橡胶/复合 | 80 | 4.4% |
| ASR | 其他 | 8 | 0.5% |
| **总计** | - | **1,800** | **100%** |

### 分类汇总

| 类别 | 质量(kg) | 占比 | 来源 |
|------|---------|------|------|
| 可重用（mPr+mDr） | 130 | 7.2% | 电机、部分内饰翻新 |
| 可回收材料（mPm+mDm+mM） | 1,470 | 81.7% | 钢铝铜+EV电池（按Battery Reg回收率）+油液 |
| 可能量回收（mTe） | 160 | 8.9% | ASR能量回收部分 |
| 处置（md） | 40 | 2.2% | 填埋 |

### 结果

- **Rcyc** = (130 + 1,470) / 1,800 = **88.9%** ✅ (≥85%)
- **Rcov** = (130 + 1,470 + 160) / 1,800 = **97.8%** ✅ (≥95%)

## BEV电池的特殊计入

按Battery Reg Art.6(23)要求，电池材料回收率：

| 材料 | 2025回收率目标 | 2030回收率目标 |
|------|--------------|--------------|
| 锂（Li） | 50% | 80% |
| 钴（Co） | 90% | 95% |
| 镍（Ni） | 90% | 95% |
| 铜（Cu） | 90% | 95% |
| 锰（Mn） | 80% | 95% |

**ISO 22628计算中的电池：**
- 2025前：按总质量×50%计入mM（保守）
- 2025-2030：按Battery Reg目标计入
- 2030后：按Battery Reg加严目标计入

## 材料分类原则

### 可重用（Reusable）

| 判定 | 示例 |
|------|------|
| 可直接或简单翻新后再使用 | 再制造发动机 |
| 有明确的二手市场或再制造商 | 变速箱、后视镜 |
| 功能满足新车标准 | 加热座椅 |

**注意：** 仅有"潜在可重用性"不够——必须有**实际的回收链**（再制造商、二手市场）。

### 可回收（Recyclable）

| 判定 | 示例 |
|------|------|
| 可通过现有技术回收为新材料 | 钢、铝、玻璃、部分塑料 |
| 经济可行 | 高价值材料（Cu、贵金属） |
| 不需新建设施 | 常规回收链 |

### 可能量回收（Recoverable）

| 判定 | 示例 |
|------|------|
| 不可材料回收但可燃烧供能 | ASR的塑料/橡胶 |
| 有足够热值 | - |
| EU有对应焚烧厂 | - |

### 不可回收（Disposal）

| 判定 | 示例 |
|------|------|
| 填埋 | 无机灰渣 |
| 无回收路径 | 某些复合材料 |
| 污染严重 | 石棉（虽已禁） |

## 常见错误

1. **过度乐观的重用率**：声明某部件"可重用"但实际无再制造商。
2. **忽略ASR中的填埋部分**：假设所有ASR能量回收，忽略20-40%填埋。
3. **电池按总质量100%计入可回收**：未按Battery Reg Art.6(23)的实际回收率。
4. **碳纤维复合材料乐观归类**：CFRP回收率很低（<20%），常只能mTe。
5. **玻璃归为mM**：夹层玻璃（PVB中间层）回收率低。
6. **轮胎的能量回收vs材料回收**：橡胶粉化材料回收率不稳定，建议归mTe。
7. **供应商数据的集合误差**：每个零部件+1%的乐观→整车+5%偏离。

## IDIS数据与计算的关系

IDIS拆解信息为ISO 22628计算提供输入：

| IDIS信息 | 用于计算 |
|---------|---------|
| 预处理步骤 | 确定mP分类 |
| 拆卸指南 | 确定mD中哪些部件可重用 |
| 材料标识 | 分选回收的前提 |
| 材料组成 | 供应商IMDS数据 |
| 有害物质位置 | 从mD中扣除（需特殊处理）|

## 合规检查清单

- [ ] 是否按ISO 22628完成了计算？
- [ ] 所有五步都有明确数据？
- [ ] Rcyc≥85%且Rcov≥95%？
- [ ] BEV电池是否按Battery Reg要求计入？
- [ ] 供应商IMDS数据是否完整（至少覆盖90%质量）？
- [ ] 重用部件是否有实际再制造商依据？
- [ ] ASR能量回收vs填埋比例是否基于实际数据？
- [ ] 计算结果是否纳入型式认证文件？
- [ ] IDIS拆解信息是否与计算一致？

## 常见不合规情况

1. **仅纸面计算合规**：数字都对，但IDIS拆解信息支撑不了计算（虚报）。
2. **EV电池未协调Battery Reg**：简单按现行平均回收率，未按Battery Reg Art.6(23)计入。
3. **CTB/CTC电池难分类**：电池与车身一体化导致"电池质量"边界模糊。
4. **Technical Service质疑**：现场抽查拆解实操与计算假设不符。
5. **新材料未评估**：新加入的碳纤维件影响Rcyc但未重新计算。

# 性能和耐久性要求

> EU 2023/1542 Art.10 + Annex IV

---

## 适用范围

- **EV电池：** 2024-08-18起需在技术文档中提供性能和耐久性参数
- **工业可充电>2kWh：** 2024-08-18起提供参数；2027-08-18起最低值强制
- **LMT电池：** 2028-08-18起最低值强制

## EV电池性能参数 — Annex IV Part A

以下参数必须在技术文档中声明：

| 参数 | 说明 | 单位 |
|------|------|------|
| 额定容量 | 标称条件下的容量 | Ah |
| 额定能量 | 标称条件下的能量 | kWh |
| 额定电压 | 标称电压 | V |
| 内阻 | 初始内阻 | mΩ |
| 循环寿命 | 在规定条件下达到80% SoH的预期循环次数 | 次 |
| 容量保持率 | 规定循环次数后的容量保持百分比 | % |
| 能量保持率 | 规定循环次数后的能量保持百分比 | % |
| 功率能力 | 不同SoC下的功率输出能力 | W |
| 预期使用寿命 | 预期的日历寿命 | 年 |
| 温度范围 | 正常工作温度范围 | °C |

## BMS数据访问 — Art.14

| 要求项 | 具体要求 |
|--------|---------|
| SoH数据开放 | BMS必须能向电池所有者和第三方（维修商、回收商）提供SoH相关数据 |
| 只读访问 | 第三方只读访问，不可修改BMS参数 |
| 数据参数 | SoC、SoH、剩余容量、总能量吞吐量等 |
| 二次利用评估 | 为电池二次利用（second life）提供足够的健康状态信息 |

## 合规检查清单

- [ ] 技术文档是否包含所有Annex IV Part A要求的性能参数？
- [ ] 循环寿命和容量保持率声明是否基于标准化测试条件？
- [ ] BMS是否开放SoH只读接口？
- [ ] 第三方（维修商、回收商）能否访问BMS健康数据？

## 常见不合规情况

1. **BMS数据锁定**：中国车企普遍将BMS数据封闭在OEM生态内，不向第三方开放。EU法规要求开放只读访问。
2. **性能参数测试条件不统一**：声明的循环寿命基于中国标准测试条件，需按EU要求的条件重新测试或换算。
3. **SoH算法不透明**：EU要求SoH数据可供第三方验证，但部分OEM的SoH算法是黑盒。

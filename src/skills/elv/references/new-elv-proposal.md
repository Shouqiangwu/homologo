# 新ELV法规提案

> EU 2023/0284(COD) — 立法进行中

---

## 提案核心变化

| 变化项 | 现行指令(2000/53/EC) | 新提案(2023/0284) |
|--------|---------------------|-------------------|
| 法规形式 | 指令（需各国转化） | **法规**（直接适用） |
| 适用范围 | M1/N1 | 可能扩展至所有M/N类 |
| 回收材料含量 | 无强制要求 | **新增**：新车中需含一定比例回收塑料/钢/铝 |
| 可回收设计 | 原则性要求 | **加强**：拆卸便利性、材料标识强制 |
| EPR（生产者延伸责任） | 基础框架 | **加强**：OEM需承担报废车辆处理费用 |
| 电池可拆卸性 | 未明确 | **新增**：EV电池必须可安全拆卸 |
| 与Battery Reg协调 | 无 | **明确协调**电池回收 |

**注意：** 该提案仍在EU立法程序中，最终文本可能有变化。建议持续跟踪。

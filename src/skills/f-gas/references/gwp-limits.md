# GWP限制和合规制冷剂速查

> EU 2024/573 F-GAS法规

---

## F-GAS法规对汽车的主要影响

EU 2024/573（新版F-GAS）主要管控HFC（氢氟碳化合物）的生产和使用配额。对汽车行业：
- MAC Directive的GWP≤150要求不变
- F-GAS法规额外限制HFC的EU市场配额（phase-down），影响R-1234yf的供应价格
- 鼓励向CO2(R-744)等天然制冷剂过渡

## 关键数值速查

| 参数 | 数值 |
|------|------|
| M1/N1车辆空调制冷剂GWP上限 | **150** |
| R-1234yf GWP | 4 |
| R-744 (CO2) GWP | 1 |
| R-134a GWP | 1,430（**不合规**） |

# 标签和标识要求

> EU 2023/1542 Art.13 + Annex VI

---

## 标签要求汇总

| 标签/标识 | 适用类别 | 强制日期 | 要求 |
|----------|---------|---------|------|
| CE标志 | 全部 | 2024-08-18 ✅ | 投放市场前标贴，可见、清晰、不可磨灭 |
| 分类收集标志 | 全部 | 2025-08-18 ✅ | 划叉轮式垃圾桶图标，≥最大面3%面积，最大5×5cm |
| 容量标签 | 全部 | 2024-08-18 ✅ | 额定容量(Ah)、标称电压(V)、能量(Wh) |
| 有害物质标识 | 含铅/镉 | 2024-08-18 ✅ | 化学符号Cd/Pb |
| 碳足迹标签 | EV/工业/LMT | EV: 2026-08-18 | 碳足迹性能等级 |
| QR码 | EV/工业>2kWh/LMT | 2027-02-18 | 链接至电池护照和符合性声明 |

## QR码内容

从2027-02-18起，QR码必须提供以下信息的访问入口：

- 符合性声明（Declaration of Conformity）
- 碳足迹声明和性能等级
- 回收材料含量声明
- 电池护照数据
- 安全数据表（SDS）
- 尽职调查报告摘要

## 合规检查清单

- [ ] CE标志是否已正确标贴？
- [ ] 分类收集标志是否符合面积要求（≥最大面3%）？
- [ ] 容量/电压/能量标签是否完整？
- [ ] 如含铅/镉，是否标注了化学符号？
- [ ] QR码是否生成并可正常扫描访问？（2027-02起）
- [ ] QR码链接的内容是否完整？

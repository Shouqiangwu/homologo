# 拆解信息提交要求

> ELV Directive 2000/53/EC Art.8 + IDIS平台

---

## 提交内容

OEM需通过IDIS平台提交以下信息：

| 信息类别 | 具体内容 |
|---------|---------|
| **车辆概述** | 车型名称、类别、生产年份、整备质量 |
| **预处理信息** | 油液排放点（燃油、冷却液、制动液等）、安全气囊位置和拆除方法 |
| **拆卸指南** | 主要可回收零部件的拆卸步骤和工具要求 |
| **材料信息** | 主要零部件的材料标识（按ISO 1043塑料标识、ISO 11469标识） |
| **有害物质位置** | 含铅/汞/镉/六价铬零部件的位置和安全处理指南 |
| **EV电池信息** | 高压系统断电步骤、电池包拆卸指南、安全注意事项 |
| **空调制冷剂** | 制冷剂类型、充注量、回收方法 |

## BEV特殊要求

| 信息 | 说明 |
|------|------|
| 高压安全 | 高压系统隔离/断电的完整步骤（橙色高压线束识别） |
| 电池拆卸 | 电池包固定点位、质量、拆卸工具、安全装备要求 |
| 电池运输 | 拆卸后电池的安全存储和运输要求（ADR危险品运输） |
| 热失控风险 | 受损电池的应急处理指南 |

## 提交流程

1. 在IDIS平台（www.idis2.com）注册OEM账号
2. 按IDIS数据格式录入车型拆解信息
3. 新车型上市6个月内完成提交
4. 车型更新（改款/新配置）时更新拆解信息

## 合规检查清单

- [ ] 是否在IDIS平台注册了OEM账号？
- [ ] 新车型是否在上市6个月内提交了拆解信息？
- [ ] 预处理信息（油液排放、安全气囊）是否完整？
- [ ] 材料标识是否按ISO 1043/11469执行？
- [ ] EV电池拆卸安全指南是否包含？
- [ ] 高压系统断电步骤是否详细？
- [ ] 有害物质位置是否标注？
- [ ] 空调制冷剂信息是否包含？

## 常见不合规情况

1. **未在IDIS注册**：中国车企首次进入EU市场，不知道IDIS平台的存在。
2. **BEV电池拆卸信息缺失**：仅提供了传统ICE车辆的拆解信息，未包含高压电池系统。
3. **信息语言问题**：拆解信息需提供英文版本（部分国家要求本地语言翻译）。
4. **延迟提交**：超过上市后6个月的期限。

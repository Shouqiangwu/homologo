# ADAS系统在用性能监控

> EU 2019/2144 + EU 2018/858 市场监督

---

## ADAS ISC的特殊挑战

与排放ISC不同，ADAS系统（AEB/ISA/ELKS等）的在用性能监控是一个新兴领域，法规框架仍在发展中。核心挑战：

| 挑战 | 说明 |
|------|------|
| 性能退化 | 传感器老化/脏污/校准偏移可能导致AEB等系统性能下降 |
| OTA更新影响 | R156 OTA更新可能改变ADAS性能特征，需评估是否影响型式认证 |
| 测试方法 | 尚无标准化的ADAS ISC测试规程（与排放ISC不同） |
| 数据来源 | 可通过OBD/远程诊断获取ADAS系统状态数据（但涉及GDPR） |

## OTA更新后的ISC

| 场景 | 要求 |
|------|------|
| 非安全相关OTA | 通常不影响型式认证 |
| 安全相关OTA（修改AEB/ISA/ELKS标定） | 需更新RXSWIN → 通知认证机构 → 可能需要ISC重新评估 |
| 性能降级OTA（如因安全问题临时降低功能） | 需通知认证机构并说明影响 |

## 合规检查清单

- [ ] 是否有ADAS系统在用性能监控计划？
- [ ] 传感器维护/校准指南是否纳入车辆用户手册？
- [ ] OTA更新是否有型式认证影响评估流程？
- [ ] 安全相关OTA后是否更新RXSWIN并通知认证机构？
- [ ] 如收到用户投诉ADAS性能问题，是否有调查和响应流程？

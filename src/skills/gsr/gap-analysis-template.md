# GSR Gap Analysis 输出模板

> 适用于EU 2019/2144通用安全法规的合规差距分析

---

## 使用说明

本模板定义了GSR领域gap analysis的输出格式。AI agent在接收到车企技术文档后，逐项比对以下检查清单，按三种状态输出结构化报告。

## 输出格式定义

### ✅ 符合（Pass）

```
### [检查项编号] [检查项名称] — ✅ 符合

**判定依据：**
法规要求：[法规编号 条款号] — [要求摘要]
→ 车企文档：《[文档名称]》第X.X节 — "[相关内容引用]"
→ 判定：满足要求，无需进一步操作。
```

### ❌ 不符合（Fail）

```
### [检查项编号] [检查项名称] — ❌ 不符合

**法规要求：**
[法规编号 条款号] — [具体要求，含数值阈值]

**车企实际情况：**
《[文档名称]》第X.X节 — "[实际参数或描述引用]"

**差距分析：**
[实际值与要求值之间的具体差距描述]

**改进建议：**
[具体改进措施，含技术修改方向和需要补充的测试]

**优先级评估：**
[🔴 阻断型式认证 / 🟡 需在认证前解决但有替代方案 / 🟢 可在认证后补充]
[说明理由]
```

### ⚠️ 信息不足（Insufficient Info）

```
### [检查项编号] [检查项名称] — ⚠️ 信息不足

**需检查的法规条款：**
[法规编号 条款号] — [要求摘要]

**信息不足原因：**
[□ 文档中完全未提及 / □ 描述模糊无法判定 / □ 数据格式不匹配 / □ 文档版本过旧]
[具体说明]

**需补充的信息：**
[具体的文档名称/测试报告类型/技术参数名称]

**建议获取方式：**
[向OEM索要什么文档 / 安排什么测试 / 咨询哪个部门]
```

# 新ELV法规提案（EU 2023/0284）

> EU 2023/0284(COD) — 2023-07-13欧委会提交，立法程序进行中

---

## 立法状态和时间线

| 阶段 | 日期 | 状态 |
|------|------|------|
| 欧委会提案 | 2023-07-13 | 已完成 |
| 欧洲议会一读 | 2024-03 | 已通过修改版 |
| 欧洲理事会立场 | 2024-06-27 | General Approach已达成 |
| Trilogue谈判 | 2024-2025 | 进行中 |
| 预计最终通过 | 2025 Q4 - 2026 Q2 | - |
| 预计强制适用日 | **2028-2030过渡期** | 生效后5-7年 |

**注意：** 最终文本可能与当前提案有变化。合规部门应订阅EUR-Lex跟踪进展。

## 从指令到法规的根本变化

| 维度 | 现行指令（2000/53/EC） | 新法规提案（2023/0284） |
|------|----------------------|------------------------|
| 法律形式 | Directive（指令，需成员国转化） | **Regulation**（法规，直接适用） |
| 适用一致性 | 27国各自转化，存在差异 | 全EU统一标准 |
| 适用范围 | M1、N1 | **扩展至M2、M3、N2、N3、O、L**（分阶段） |
| 管辖主体 | 成员国执法 | EU+成员国双层执法 |

## 核心新增要求

### 1. 回收材料含量（Recycled Content）— 最重磅

| 材料 | 最低回收含量（新车含量） | 生效时间 |
|------|------------------------|---------|
| 塑料 | **25%**（其中至少25%来自ELV） | 生效后6年 |
| 塑料（长期） | **30%** | 生效后10年 |
| 钢 | 讨论中（议会提议20%，理事会未定） | 生效后6年 |
| 铝 | 讨论中（议会提议10%） | 生效后6年 |

**对供应链的影响：**
- 塑料供应商需建立回收料（PCR）采购链
- 部分要求"close loop"：来源必须是汽车ELV，不能是其他消费品塑料
- 成本上升，初期可能+20-30%

### 2. 循环设计要求（Circular Design）

| 要求 | 说明 |
|------|------|
| **拆卸便利性** | 关键部件必须能在X分钟内无损拆卸（电池、电机、ADAS模块） |
| **零部件标识** | 塑料>100g需按ISO 1043标识，金属合金标识成分 |
| **模块化设计** | 避免粘胶集成式设计，鼓励机械连接 |
| **软件与硬件解耦** | 报废车辆的ADAS软件可被安全擦除，硬件可重用 |

### 3. 电池可拆卸性（协调Battery Reg Art.11）

| 要求 | 说明 |
|------|------|
| 电池包结构可拆 | 禁止完全一体化不可拆电池包 |
| 电芯级可拆（长期） | 2030后电芯层级的可拆卸性 |
| 拆卸工具通用 | 不得使用专用工具 |
| 热管理系统不得粘结 | 冷却液管路可安全分离 |

**对CTB/CTC设计的冲击：** 特斯拉/比亚迪的电池集成车身/底盘设计可能需重新评估。

### 4. 生产者延伸责任（Extended Producer Responsibility, EPR）

| 现行 | 新提案 |
|------|--------|
| 模糊的生产者责任（各国不同） | **强制EPR**：OEM承担ELV处理费用 |
| 按国家组织 | 建立全EU的EPR登记和资金池 |
| 回收成本由拆解方承担 | OEM按注册量缴费 |

**预计成本：** 每辆新车EPR费用约€100-300，按车辆全生命周期环境成本定价。

### 5. 材料护照（Material Passport）

| 要求 | 说明 |
|------|------|
| 建立类似Battery Passport的整车材料护照 | 数字化记录所有主要材料来源和组成 |
| 访问权限 | OEM、拆解企业、市场监督、消费者（部分） |
| 互操作 | 与Battery Passport和IDIS协调 |

### 6. 有害物质清单扩展

除现行铅/汞/镉/六价铬外，可能新增：

| 物质 | 状态 |
|------|------|
| 六溴环十二烷(HBCDD) | 提议加入 |
| 双酚A(BPA) | 评估中 |
| PFAS类（广谱） | 受REACH影响，可能纳入ELV |
| 特定阻燃剂 | 评估中 |

### 7. 可回收率目标提升

| 指标 | 现行 | 新提案 |
|------|------|--------|
| Rcyc | ≥85% | ≥85%（维持）+ 电池单独要求 |
| Rcov | ≥95% | ≥95%（维持） |
| 新增：实际回收率 | - | 要求ELV处理设施实际达到Rcyc的90% |

**区别：** 现行只要设计达到Rcyc≥85%，新提案要求实际处理也达到（拆解企业需认证）。

### 8. 执法与处罚

| 项目 | 新要求 |
|------|--------|
| 违规罚款 | 按营收比例，类似GDPR机制（最高4%全球营收） |
| 市场监督权限 | 扩大欧委会直接检查权 |
| OEM自检义务 | 定期提交合规报告 |

## 对中国车企的战略影响

### 短期（2025-2028 过渡期）

| 影响 | 应对 |
|------|------|
| 回收料供应链建设 | 在EU建立PCR塑料采购通道 |
| 电池可拆卸设计 | 重新评估CTB/CTC设计可行性 |
| EPR费用预算 | 每辆车财务模型加入€100-300 |
| 材料护照系统 | IMDS基础上扩展 |

### 长期（2030后）

| 变化 | 含义 |
|------|------|
| 全欧统一合规 | 不再有成员国差异 |
| 循环经济强制 | 线性经济设计无法准入EU |
| 软件定义车辆挑战 | 报废时软件可擦除能力成硬性要求 |
| 中国模式转型 | 国内一体化降本逻辑可能与EU循环设计冲突 |

## 常见应对误区

1. **"等法规通过再应对"**：塑料回收料供应链和电池可拆卸设计需≥3-5年，等待即错失窗口。
2. **低估回收料成本**：PCR塑料初期比原生料贵20-40%，影响财务模型。
3. **电池集成设计固守**：CTB/CTC虽降本但可能不兼容新要求，需双轨设计。
4. **EPR仅作财务影响看**：EPR建立了向后追责机制，OEM真正成为全生命周期责任主体。
5. **忽略中国供应商适配**：中国塑料供应商无EU ELV PCR供给能力，需提前布局。

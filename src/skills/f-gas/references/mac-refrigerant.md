# MAC Directive 合规详解

> Directive 2006/40/EC + 实施法规 EC 706/2007

---

## MAC Directive的核心要求

MAC (Mobile Air Conditioning) Directive 2006/40/EC针对M1和N1 Class I（≤1305 kg参考质量）车辆的空调系统设定三大要求：

1. **GWP限制**：制冷剂GWP≤150
2. **泄漏率限制**：年泄漏率上限
3. **型式认证要求**：空调系统需通过型式认证

## 适用范围详解

| 车辆类别 | 参考质量 | MAC适用 | F-GAS适用 |
|---------|---------|---------|----------|
| M1（乘用车） | 全部 | ✅ | ✅ |
| N1 Class I | ≤1305 kg | ✅ | ✅ |
| N1 Class II | 1305-1760 kg | ❌（但F-GAS适用） | ✅ |
| N1 Class III | >1760 kg | ❌ | ✅ |
| M2/M3（公交） | - | ❌ | ✅ |
| N2/N3（卡车） | - | ❌ | ✅ |

## GWP限制 — 详细规则

### GWP≤150 的适用范围

| 适用 | 不适用 |
|------|--------|
| 新车型型式认证（2011起） | 现有型式认证的维持 |
| 所有新注册车（2017起） | 售后维修的补充充注（可用原规格） |
| 工厂充注的主系统制冷剂 | 小型辅助系统（某些双回路系统） |

**重要细节：** 若车辆原设计使用R-134a（2017前出厂），该车售后维修仍可补充R-134a。但EU市场上新销售的任何M1/N1必须≥150 GWP。

### GWP值的引用标准

EU法规目前采用IPCC AR5的GWP-100值：

| 制冷剂 | AR5 GWP | AR4 GWP（历史） |
|--------|---------|----------------|
| R-134a | 1,430 | 1,300 |
| R-1234yf | 4 | 4 |
| R-744 | 1 | 1 |
| R-152a | 138 | 124 |

**注意：** IPCC AR6发布后数值略有调整（如R-134a AR6=1,530），但EU法规未来是否更新尚未明确。

## 泄漏率限制

### 限值

MAC Directive Annex III设定年泄漏率上限：

| 系统类型 | 最大年泄漏率 |
|---------|------------|
| 单蒸发器系统 | **40 g/年** |
| 双蒸发器系统 | **60 g/年** |

### 测试方法 — ECE R82 (SAE J2763)

泄漏率测试程序：

| 步骤 | 要求 |
|------|------|
| 1. 系统充注 | 按OEM规格充注制冷剂 |
| 2. 环境温度 | 40°C ± 5°C（测试箱） |
| 3. 测试时长 | **24小时**（连续） |
| 4. 测量方法 | 称重法或SF6示踪气体法 |
| 5. 发动机状态 | 系统开启运转 + 关闭静置组合测试 |
| 6. 结果外推 | 24小时测量值外推至年度泄漏率 |

### 实际泄漏率数据

| 系统类型 | 典型泄漏率（良好设计） |
|---------|----------------------|
| 单蒸发器R-1234yf | 10-20 g/年 |
| 双蒸发器R-1234yf（SUV/MPV） | 20-40 g/年 |
| R-744系统（高压） | 5-15 g/年（系统设计优） |

## 型式认证流程

### 适用法规

| 法规 | 要求 |
|------|------|
| EU 2018/858 Art.30 | 需获得EU型式认证 |
| ECE R10 | 电磁兼容性（含A/C控制） |
| Directive 2006/40/EC | MAC专项 |
| EU 706/2007 | MAC实施细则 |

### 型式认证文件清单

| 文件 | 内容 |
|------|------|
| **Information Document** | 空调系统技术规格（制冷剂、充注量、结构图） |
| **Test Report** | 泄漏率测试报告（ECE R82） |
| **Components List** | 压缩机、冷凝器、蒸发器、膨胀阀/节流阀规格 |
| **Compliance Statement** | GWP≤150声明 |
| **Label Design** | 制冷剂标签设计 |

### Technical Service和Certification Body

- Technical Service执行测试（TÜV、DEKRA、IDIADA等）
- 认证机构颁发型式认证号（如e1 for Germany）
- 型式认证号需在车辆一致性证书（CoC）中引用

## 标签要求

### 制冷剂标识（EN 378-1）

空调系统上必须有永久性标签，内容：

| 内容 | 示例 |
|------|------|
| 制冷剂代号 | R-1234yf |
| 化学名称 | HFO-1234yf |
| 充注量 | 550 g |
| GWP值 | 4 |
| 警告符号 | 可燃性（A2L用橙色） |

### 标签位置

- 发动机舱内显眼位置
- 乘员舱不得放置（安全原因）
- 字体高度≥3 mm，颜色对比鲜明

### 中国车企常见标签问题

- 标签仅中文（需英文）
- 仅标R-134a（遗留）
- 未标充注量
- 未标GWP

## 维修后的合规

### 充注制冷剂溯源

每次售后充注需记录：

| 数据 | 要求 |
|------|------|
| 车辆VIN | 关联车辆 |
| 充注量 | 精确到5g |
| 制冷剂批次 | 来源证明 |
| 技术员认证号 | 按F-GAS Art.10 |
| 回收量（如有抽出） | 用于泄漏评估 |

### EU F-GAS登记

维修网络需登记在EU F-GAS Registry（通过FAPE或本国系统），记录：
- 年度采购的制冷剂总量
- 年度销售到车辆的充注量
- 年度回收的制冷剂量

## 合规检查清单

- [ ] 空调系统是否获得EU型式认证？
- [ ] 型式认证涵盖ECE R82泄漏率测试？
- [ ] 制冷剂GWP≤150？
- [ ] 充注量符合原设计？
- [ ] 泄漏率≤40g/年（单蒸发器）或≤60g/年（双蒸发器）？
- [ ] 空调系统有永久性标签（含制冷剂、充注量、GWP、警告）？
- [ ] 标签提供英文（和当地语言）？
- [ ] 售后维修网络技术员持F-GAS Art.10认证？
- [ ] 售后制冷剂使用有登记系统？

## 常见不合规情况

1. **泄漏率勉强通过**：实验室测试40 g/年，实际使用中因部件老化泄漏加剧——建议设计余量至20-30 g/年。
2. **型式认证与量产不一致**：型式认证用某批次压缩机，量产改其他供应商未更新认证。
3. **标签双语未落实**：仅中文标签不合规。
4. **R-1234yf可燃性标识缺失**：A2L类别警告未添加。
5. **售后网络未配F-GAS认证技师**：部分地区独立维修店无资质。
6. **充注量公差过大**：OEM规格±10%，实际批次波动15%，边缘不合规。
7. **BEV热泵系统标签遗漏**：仅在发动机舱搜索标签，忽略BEV的前舱热泵单元。

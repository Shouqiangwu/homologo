# 排放在用车符合性测试

> EU 2018/1832 + EU 2018/858 Art.52

---

## 排放ISC要求

| 要求项 | 具体要求 |
|--------|---------|
| OEM义务 | 制造商必须持续监控已售车辆的排放一致性 |
| 测试对象 | 从市场上抽取的在用车辆 |
| 测试方法 | 按照型式认证时使用的测试规程（WLTP/RDE） |
| 合格判定 | 在用车排放值不得超过型式认证值的规定倍数（conformity factor） |
| 报告义务 | OEM需向型式认证机构提交年度ISC报告 |
| 不合规后果 | 可能导致型式认证撤销、强制召回 |

## 实际排放监控（OBFCM）

EU 2021/392要求新车配备OBFCM（On-Board Fuel Consumption Monitoring），记录实际油耗/能耗：
- 数据在年检时读取
- 用于比较型式认证CO2值与实际使用CO2值
- 差异过大可能触发调查

## 召回程序

| 场景 | 触发者 | 流程 |
|------|--------|------|
| OEM自主召回 | OEM发现合规问题 | 通知认证机构→制定纠正方案→执行召回 |
| 监管强制召回 | 市场监督机构发现问题 | 通知OEM→限期整改→如不整改可撤销认证 |

## 合规检查清单

- [ ] 是否建立了ISC监控计划？
- [ ] 是否定期从市场抽取车辆进行排放测试？
- [ ] OBFCM系统是否已安装并正常工作？
- [ ] 是否向认证机构提交年度ISC报告？
- [ ] 是否有召回应急预案？

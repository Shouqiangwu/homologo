# 制冷剂回收与处置

> EU 2024/573 Art.5/Art.8 + ELV Directive 2000/53/EC Annex I + F-GAS Art.22

---

## 三个回收场景

车辆生命周期中制冷剂需要被回收/处置的三个场景：

| 场景 | 时机 | 主体 |
|------|------|------|
| **售后维修** | 维修时需排空系统 | 维修技师（F-GAS Art.10认证） |
| **车辆报废（ELV）** | 拆解预处理阶段 | 授权处理设施（ATF） |
| **报废制冷剂处理** | 回收后不可重用时 | 专业处理企业 |

## 维修场景的回收

### 法规要求

- F-GAS Art.8(2)：含氟制冷剂维修时不得故意释放到大气
- 必须使用回收设备（recovery unit）进行回收
- 技术员必须持F-GAS Art.10认证

### 回收流程

```
Step 1: 连接回收机至A/C维修端口（低压/高压）
Step 2: 启动回收机，抽取气态+液态制冷剂
Step 3: 称重确认回收量
Step 4: 检查回收机油水分离器（避免油污染）
Step 5: 回收制冷剂贮存罐（专用钢瓶）
```

### 回收率要求

专业回收设备需达到EN 378或类似标准：
- 回收率≥95%（质量）
- 残留在系统中≤5%

### 回收后处理

| 制冷剂状态 | 处理方式 |
|----------|---------|
| 清洁（无污染） | 可现场或集中再充注（reuse） |
| 水分/油污染 | 送专业再生（reclaim）设施 |
| 混合制冷剂 | 不可直接reuse，必须reclaim或destruct |
| 无法再生 | 热销毁（incineration）或化学销毁 |

## ELV场景的回收

### ELV Directive Annex I的要求

ELV车辆预处理阶段必须拆除/回收：
- 燃油、机油、转向液、制动液、冷却液
- **A/C制冷剂**（明确列出）
- 蓄电池、轮胎、催化转化器

### 操作流程

```
车辆进入ATF → 识别车辆信息（VIN, 年款, 制冷剂类型）
    ↓
定位A/C维修端口 → 连接专业回收机
    ↓
抽空A/C系统 → 记录回收量
    ↓
制冷剂暂存 → 按类型分类（不可混）
    ↓
运送至制冷剂处理企业 → reclaim或destruct
```

### ATF的义务

| 义务 | 要求 |
|------|------|
| 拆解企业资质 | 按ELV Directive获得国家级或地方授权 |
| 设备要求 | 配备符合EN 378的回收设备 |
| 技师资质 | F-GAS Art.10认证 |
| 记录 | 保留每辆车的回收量、处理记录5年 |
| 报告 | 年度向监管机构报告处理量 |

## 报废制冷剂处理

### 再生（Reclamation）

符合新制冷剂规格的处理：
- 脱水、过滤、蒸馏
- 质量检测（纯度≥99.5%）
- 可作为"reclaimed refrigerant"重新销售

**经济性：** R-1234yf reclaim后价格约为新制冷剂的70-80%。

### 销毁（Destruction）

不可再生时：

| 技术 | 原理 | 销毁效率(DRE) |
|------|------|-------------|
| 高温焚烧 | 1,100°C+，配备酸气处理 | ≥99.99% |
| 等离子弧 | 高温等离子分解 | ≥99.99% |
| 催化氧化 | 催化剂辅助氧化 | ≥99.9% |

**合规要求：** 销毁设施需通过EU或成员国认证，DRE（销毁和去除效率）≥99.99%。

### 销毁的成本

R-1234yf销毁成本约€10-20/kg，低于reclaim的经济价值——所以行业优先reclaim。

## OEM的责任（新ELV提案强化）

### 现行

- 通过IDIS提供拆解信息（含制冷剂回收步骤）
- 无财务责任（ELV处理费由ATF承担）

### 新ELV提案

- 强制EPR：OEM承担全生命周期制冷剂处理费
- 按新车制冷剂充注量计算EPR费用
- 加速向天然制冷剂（R-744, R-290）转型（低处理成本）

## 对中国车企的实操要点

### 拆解信息提供

在IDIS中必须包含：

| 信息 | 说明 |
|------|------|
| 制冷剂类型 | R-1234yf / R-744 / R-290 |
| 充注量 | 精确到5g |
| 维修端口位置 | 低压/高压端口在发动机舱的图示位置 |
| 回收顺序 | 先低压后高压，或OEM指定顺序 |
| 安全警告 | 可燃性（R-1234yf A2L）、窒息风险（R-744密闭空间） |
| 特殊工具 | 如R-744需高压适配器 |

### 售后维修网络

- EU所有授权维修网点必须有F-GAS认证技师
- 独立维修店不持认证不得处理制冷剂
- 中国车企进入EU需支持经销商网络配置

### 制冷剂采购合规

| 要点 | 说明 |
|------|------|
| 供应商F-GAS配额 | R-1234yf原料涉及HFO前体，供应商需有EU HFC配额 |
| 进口报告 | 进口R-1234yf到EU需登记 |
| 批次追溯 | 维修用制冷剂需有溯源 |

## 合规检查清单

- [ ] 制冷剂类型是否明确且在IDIS中标注？
- [ ] 充注量是否精确记录？
- [ ] 维修端口位置在拆解手册中是否清楚？
- [ ] 售后维修网络技术员是否持F-GAS认证？
- [ ] 维修回收设备是否符合EN 378？
- [ ] 中国出口的制冷剂是否通过EU授权进口商？
- [ ] 报废阶段ATF的回收责任和流程是否在拆解手册中说明？
- [ ] 是否跟踪了新ELV提案对EPR费用的影响？

## 常见不合规情况

1. **维修技师无F-GAS认证**：偏远地区维修点使用未认证技师处理制冷剂。
2. **回收率低**：老旧回收设备导致大量制冷剂残留释放（实际回收率<85%）。
3. **制冷剂混用**：维修中误将R-134a回收罐用于R-1234yf收集，无法reclaim只能destruct。
4. **IDIS信息简陋**：仅标"标准A/C"，未说明维修端口位置和特殊工具要求。
5. **R-744高压系统误操作**：不了解85-120 bar系统的ATF人员使用普通设备导致危险。
6. **报废库存制冷剂违规**：OEM/经销商库存的老旧R-134a未按F-GAS Art.22报告。
7. **跨境转运未报备**：从EU外进口R-1234yf维修用量未登记。

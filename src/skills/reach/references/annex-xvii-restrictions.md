# REACH Annex XVII 限制物质（汽车应用）

> EU 1907/2006 Annex XVII + 多项修订法规

---

## Annex XVII 机制

REACH Annex XVII列出了在EU市场对特定物质的**使用限制**。与SVHC候选清单（仅申报义务）不同，Annex XVII限制物质在达到特定条件后**禁止使用或投放市场**。截至2025年，Annex XVII约含**80条条目**（Entries），约一半直接或间接影响汽车行业。

| SVHC候选清单 | Annex XIV授权清单 | Annex XVII限制清单 |
|-------------|------------------|-------------------|
| 申报义务（Art.33, SCIP） | 使用需授权 | 直接限制/禁止 |
| ~240种物质（2025） | ~60种物质 | 约80条Entries |

## 汽车行业最相关的Annex XVII条目

### Entry 23 — 镉及镉化合物

| 限制 | 适用 |
|------|------|
| 塑料中镉含量≤0.01%（w/w） | 所有车用塑料件 |
| 电镀层中的镉（工业电镀除外） | 紧固件、装饰件 |
| 电池中镉 | ≤0.002%（2016起），Battery Reg Art.6进一步限制 |
| 焊料中镉 | ≤0.01% |

### Entry 27 — 镍及其化合物（致敏性）

| 限制 | 适用 |
|------|------|
| 与皮肤长时间直接接触物品中镍释放率 | ≤0.5 μg/cm²/周 |
| 汽车涉及 | 方向盘装饰件、门把手、仪表板按钮（如含镍） |

### Entry 47 — 铬(VI)化合物

| 限制 | 适用 |
|------|------|
| 皮革制品中Cr(VI) | ≤3 mg/kg |
| 汽车涉及 | 皮革座椅、方向盘皮革包裹 |

### Entry 50 — 多环芳烃(PAHs)

| 限制 | 适用 |
|------|------|
| 橡胶或塑料件中8种PAHs总量 | ≤1 mg/kg（消费者接触件） |
| 汽车涉及 | 轮胎（填充油）、减震橡胶件、密封条、方向盘橡胶 |
| 轮胎特别条款 | 填充油中BaP（苯并[a]芘）等≤1 mg/kg总和 |

### Entry 51 — 邻苯二甲酸酯（DEHP, DBP, BBP, DIBP）

| 限制 | 适用 |
|------|------|
| 塑料化物品中4种邻苯二甲酸酯总量 | ≤0.1%（w/w） |
| 汽车涉及 | PVC线束护套、密封条、内饰软塑料、脚垫 |
| 生效日 | 2020-07-07起全面生效 |

### Entry 52 — 其他邻苯二甲酸酯扩展

欧委会持续扩展邻苯二甲酸酯限制至更多物质（DINP, DIDP等）——需持续监控。

### Entry 63 — 铅及铅化合物（消费品）

| 限制 | 适用 |
|------|------|
| 消费品中可迁移铅 | ≤0.05%（某些接触件更严） |
| 汽车涉及 | 乘员舱内饰（座椅、方向盘、门板） |
| 豁免 | ELV Annex II豁免项目（与REACH协调） |

### Entry 68 — PFOA及相关物质

| 限制 | 适用 |
|------|------|
| PFOA及盐类 | 制造/投放市场受限 |
| 汽车涉及 | 防水涂层（座椅面料、织物地毯） |
| 生效日 | 2020-07-04起 |

### Entry 72 — CMR物质（致癌/诱变/生殖毒性）

| 限制 | 适用 |
|------|------|
| 纺织品中33种CMR物质 | ≤特定浓度 |
| 汽车涉及 | 座椅面料、安全带织带、地毯 |

### Entry 76 — CMR在消费品中（扩展清单）

持续扩展的消费品CMR限制，与汽车内饰和乘员接触件相关。

### Entry 78 — 石棉

| 限制 | 适用 |
|------|------|
| 石棉纤维及含石棉物品 | 全面禁止 |
| 汽车涉及 | 历史上在摩擦材料（刹车片、离合器片）中使用，现已完全禁止 |
| 注意 | 某些中国进口零部件历史上可能含石棉，需严格筛查 |

### Entry 28/29/30 — CMR物质向消费者销售

限制特定CMR物质向公众销售，影响售后维修耗材（冷却液、除冰剂等）。

## 关键时间线（2020-2026修订）

| 日期 | 变更 |
|------|------|
| 2020-07-07 | Entry 51（4种邻苯二甲酸酯）全面生效 |
| 2020-07-04 | Entry 68（PFOA）生效 |
| 2023-02-25 | Entry 50（PAHs）扩展至消费品 |
| 2024（预计） | 全氟/多氟化合物（PFAS）广泛限制立法 — 汽车涂层和密封件重大影响 |
| 2025 | 微塑料限制（EU 2023/2055）影响轮胎填充料 |

## PFAS重点关注（2026-2030）

欧委会及德/荷/丹/瑞/挪联合提案：限制**10,000+种PFAS物质**。汽车行业受影响面：

| 应用 | 风险等级 | 替代可行性 |
|------|---------|-----------|
| 防水防污涂层 | 高 | 中（替代品性能略差） |
| 氟橡胶密封件（O-rings、油封） | 高 | 低（高温高压应用难替代） |
| 含氟润滑油 | 中 | 中 |
| 含氟塑料（PTFE, PVDF, FKM） | 高 | 低至中 |
| 燃料电池膜 | 极高 | 目前无替代 |

**时间线预期：**
- 2025 立法通过
- 2026-2030 过渡期
- 2030起强制合规

## 与其他法规的交叉

| Annex XVII限制 | 交叉法规 | 说明 |
|---------------|---------|------|
| Entry 23（镉） | ELV Annex II + Battery Reg | ELV有豁免但Annex XVII是底线 |
| Entry 51（邻苯） | ELV Annex II | ELV未限邻苯，REACH补足 |
| Entry 63（铅） | ELV Annex II | ELV有豁免铅，Annex XVII对消费品更严 |
| Entry 50（PAHs） | REACH SVHC | PAHs部分物质同时是SVHC |

## 合规检查清单

- [ ] 是否筛查了车辆所有塑料件的镉含量（Entry 23）？
- [ ] 皮革件是否通过Cr(VI)测试（Entry 47）？
- [ ] 轮胎及橡胶件PAHs是否合规（Entry 50）？
- [ ] PVC线束和密封条是否替代了DEHP等邻苯（Entry 51）？
- [ ] 座椅防水涂层是否不含PFOA（Entry 68）？
- [ ] 内饰纺织品CMR物质是否筛查（Entry 72/76）？
- [ ] 摩擦材料是否严格无石棉（Entry 78）？
- [ ] 是否为PFAS未来限制建立了替代方案研究？
- [ ] 供应商合规声明是否包含Annex XVII所有相关条目？

## 常见不合规情况

1. **邻苯限制理解为仅玩具法规**：许多供应商认为Entry 51只针对儿童玩具，忽略了汽车内饰同样适用。
2. **老旧供应商产品含PAHs**：历史采购的橡胶件使用高PAH填充油，2023年扩展到消费品后才发现。
3. **PFOA替代不彻底**：使用C6/C8 PFOA已禁，但用C4/C6替代物（仍是PFAS）也将被限制。
4. **石棉污染源不受控**：从中国采购的低端摩擦材料偶有石棉污染历史，需批次检测。
5. **未监控PFAS立法进度**：2030强制大限将至，氟橡胶密封件研发周期需≥5年。

# REACH 授权制度（Annex XIV）

> EU 1907/2006 Title VII + Annex XIV

---

## 授权制度的定位

REACH Title VII的授权制度（Authorization）针对**极高度关注物质**（SVHC中最严重的一类），要求使用者在特定**日落日期（Sunset Date）**之后若仍想使用，必须向ECHA申请并获得授权。未获授权则禁止使用。

```
SVHC候选清单 (~240种)
        ↓ 优先化
    Annex XIV授权清单 (~60种)
        ↓ 日落日后
    无授权→禁止使用
```

## 汽车行业最相关的Annex XIV物质

### 铬酸（Chromium Trioxide, CAS 1333-82-0）

| 参数 | 数值 |
|------|------|
| 日落日（Sunset Date） | 2017-09-21 |
| 适用 | 电镀、阳极氧化（硬铬层） |
| 汽车涉及 | 底盘件、紧固件、活塞环、制动盘、装饰镀层 |
| 授权持有人 | CTAC/CTACSub Consortium（行业联盟） |
| 现状 | 授权有效期通常延至2024-2028 |

**对中国供应链的影响：** 许多中国供应商仍使用六价铬电镀，出口EU必须确认客户的下游授权覆盖，或使用三价铬替代工艺。

### 铬酸盐（锶铬酸盐、铅铬酸盐等）

| 物质 | 应用 |
|------|------|
| 锶铬酸盐 | 航空航天防腐涂层（少量汽车） |
| 铅铬酸盐 | 黄色/橙色颜料（已几乎完全退出汽车涂装） |
| 锌铬酸盐 | 防腐底漆 |

### 邻苯二甲酸酯类（DEHP, DBP, BBP, DIBP）

| 参数 | 数值 |
|------|------|
| 日落日 | 2015-02-21（DEHP, DBP, BBP, DIBP） |
| 汽车涉及 | PVC软质塑料（线束护套、密封条、内饰） |
| 豁免 | 食品接触、医疗用途等——汽车无豁免 |
| 现状 | 已广泛替代为DINCH、DOTP等 |

**与Annex XVII的关系：** Annex XIV是"不得使用"，Annex XVII Entry 51是"含量≤0.1%"——授权到期后Annex XIV优先适用。

### 三氧化二硼（Boric acid/Borates）

| 参数 | 数值 |
|------|------|
| 日落日 | 2021（已过） |
| 汽车涉及 | 玻璃纤维、复合材料增强剂 |
| 影响 | GFRP部件（保险杠骨架、电池壳）需评估 |

### TCEP等阻燃剂

| 物质 | 应用 |
|------|------|
| TCEP（磷酸三氯乙酯） | 聚氨酯泡沫阻燃（座椅、内饰） |
| 替代品 | TCPP等（但部分也在评估中） |

### MOCA（芳香胺固化剂）

| 参数 | 数值 |
|------|------|
| 日落日 | 2017-11-22 |
| 汽车涉及 | 聚氨酯弹性体固化剂（轮胎、减震件） |

## 授权申请流程

### 申请人

- 使用物质的下游用户可申请
- 行业联盟（Consortium）代表批量申请是主流方式
- 中国OEM通常通过EU进口商或EU代表申请

### 申请成本

| 项目 | 费用 |
|------|------|
| ECHA申请费 | €50,000-€100,000（单次申请） |
| 化学安全报告（CSR） | 数十万欧元（咨询费） |
| 替代方案分析（AoA） | 数十万欧元 |
| 社会经济分析（SEA） | 数十万欧元 |
| 总成本 | 约€500,000-€2,000,000/次 |

### 审批时间

- ECHA科学委员会评议：12-18个月
- 欧委会决定：6-12个月
- 授权期通常：4-12年（取决于替代可行性）

## 下游用户（Downstream User, DU）义务

若使用Annex XIV物质而未自行申请授权：

1. **使用已授权物质**：在授权持有人的授权范围内使用（需接收authorization number和使用条件）
2. **向ECHA通报**：Art.66(1)，使用授权物质后3个月内通报
3. **传达至下游**：向下游客户传达授权条件（Art.66(2)）

**对中国OEM出口EU车辆的含义：**
- 若车辆零部件含Annex XIV物质 → EU进口商/EU代表需确认供应链授权有效性
- 零部件供应商需提供授权号和使用条件
- 未覆盖的使用属违法

## 与SVHC通报的关系

| 阶段 | 义务 |
|------|------|
| 入SVHC候选清单 | Art.33供应链沟通 + Art.7(2) ECHA通报 + SCIP |
| 入Annex XIV | 以上 + 授权义务 |
| 日落日后 | 无授权禁止使用 |
| 已授权使用 | 仍需SCIP通报和下游沟通 |

## 替代策略

### 铬酸(VI)电镀替代

| 替代工艺 | 性能 | 应用情况 |
|---------|------|---------|
| 三价铬电镀（Cr3+） | 装饰铬可替代，硬铬替代性能不稳定 | 装饰件已主流替代 |
| 锌镍合金 | 防腐性好 | 紧固件替代 |
| 物理气相沉积（PVD） | 高性能但成本高 | 高端部件 |
| 高速火焰喷涂（HVOF） | 硬铬替代 | 活塞杆等 |

### 邻苯二甲酸酯替代

| 替代品 | 优缺点 |
|--------|-------|
| DINCH | 已主流，性能相当 |
| DOTP | 已主流 |
| 柠檬酸三丁酯（TBC） | 生物基，适用于食品接触 |

## 合规检查清单

- [ ] 是否筛查了供应链中所有Annex XIV物质的使用？
- [ ] 含铬酸(VI)电镀的零部件是否有有效授权号？
- [ ] PVC软件中DEHP等邻苯二甲酸酯是否已替代？
- [ ] 下游使用是否向ECHA做Art.66通报？
- [ ] 授权到期日是否监控？
- [ ] 是否有替代工艺的研发计划？
- [ ] 供应商合规声明是否包含Annex XIV物质状态？

## 常见不合规情况

1. **六价铬电镀未追踪授权号**：中国电镀厂虽在CTACSub覆盖范围内但未在出厂文件中标注授权号。
2. **DEHP在PVC线束中残留**：日落日后仍使用库存PVC颗粒。
3. **Art.66通报遗漏**：下游用户不知有通报义务，使用后3个月未通报。
4. **替代工艺不满足OEM技术要求**：三价铬硬度/耐磨性略逊，未通过OEM零部件规范。
5. **授权到期未及时续申**：持有人未续申→该物质进入强制退出状态，下游使用违法。

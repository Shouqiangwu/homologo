# 四种有害物质限制（铅/汞/镉/六价铬）

> ELV Directive 2000/53/EC Art.4(2) + Annex II

---

## 法规基础

ELV Art.4(2)(a)规定：自2003-07-01起，车辆材料和零部件中**不得使用**铅、汞、镉、六价铬，Annex II列出的豁免除外。

**适用范围：** M1和N1类整车所有材料和零部件（含售后备件）。

## 四种物质详细限制

### 铅（Pb）

| 项目 | 要求 |
|------|------|
| 默认限值 | 禁用 |
| 豁免依据 | Annex II共约20条豁免（含义车）|
| 典型应用（有豁免） | 钢材合金（≤0.35%）、焊料、压电陶瓷、玻璃 |
| 典型应用（无豁免已禁） | 减震器中铅、涂料颜料、PVC稳定剂、车身熔融涂层 |
| 检测方法 | X射线荧光(XRF)、ICP-OES、ICP-MS |

### 汞（Hg）

| 项目 | 要求 |
|------|------|
| 默认限值 | 禁用 |
| 豁免依据 | 早期有HID氙灯/荧光管豁免，基本全部到期 |
| 现状 | 汽车中已完全淘汰 |
| 残留风险 | 极少数早期HID氙灯存量（非整车主流） |

### 镉（Cd）

| 项目 | 要求 |
|------|------|
| 默认限值 | 禁用 |
| 豁免依据 | Annex II条目10/11基本已到期 |
| 典型应用（已禁） | NiCd电池、镉颜料、含镉焊料、电镀层 |
| REACH交叉 | Annex XVII Entry 23规定塑料中镉≤0.01% |
| 检测方法 | XRF、ICP-OES |

### 六价铬 Cr(VI)

| 项目 | 要求 |
|------|------|
| 默认限值 | 禁用 |
| 豁免依据 | 2007后主要豁免已到期 |
| 典型应用（已禁） | 防腐钝化层、车身磷化后处理 |
| 主流替代 | 三价铬（Cr+3）钝化 |
| REACH交叉 | SVHC + Annex XIV授权物质（铬酸） |
| 检测方法 | IEC 62321-7-2（沸水萃取法）、pH显色反应 |

## Annex II豁免核查流程

```
零部件使用上述物质
    ↓
查Annex II最新版（必须2023版）
    ↓
匹配具体应用条目
    ↓
├─ 有匹配 → 验证：
│  ├─ 含量是否≤限值？
│  ├─ 到期日是否未至？
│  └─ 标识要求是否满足？
│
└─ 无匹配 → 不合规，需设计变更
```

## 浓度测算原则

- 限值按**均质材料**（homogeneous material）计算
- "均质材料"指机械方法不可分离的材料（如一根铝合金线、一块塑料、一层涂层）
- 同一零部件内不同均质材料分别计算

**示例：** 一个紧固件若镀锌层中Cr(VI)超标，但本体钢材合规——仍属Cr(VI)超标（镀层是独立均质材料）。

## 新ELV提案（EU 2023/0284）的变化

| 变化 | 说明 |
|------|------|
| 新增物质 | 可能增加对六溴环十二烷(HBCDD)、双酚A(BPA)等的限制 |
| 豁免审查机制 | 每5年强制审查Annex II，推动持续收紧 |
| 与REACH深度整合 | REACH Annex XIV物质自动进入ELV禁用 |
| 成分数据库 | OEM需建立完整材料数据库（类似IMDS但合规强制） |

## 合规数据流

```
供应商 → IMDS数据录入
    ↓
OEM汇总 → 检查是否含Annex II四种物质
    ↓
├─ 有 → 查是否有Annex II豁免
│  ├─ 无豁免 → 要求供应商整改
│  └─ 有豁免 → 记录豁免号，监控到期日
│
└─ 无 → 记录为"clean"
    ↓
型式认证文件：有害物质合规声明
```

## 供应商合规声明（Declaration of Compliance）

OEM应要求每个零部件供应商提供DoC，内容至少包括：

| 要素 | 说明 |
|------|------|
| 零部件唯一标识 | Part Number + 版本号 |
| 材料清单 | 均质材料及其重量占比 |
| 四种物质声明 | 是否含有，含量，对应Annex II豁免号 |
| 其他物质声明 | SVHC >0.1%的物质清单 |
| 测试方法 | 使用的分析方法 |
| 有效期 | 通常1-2年，到期需更新 |
| 签章 | 供应商合规负责人签字 |

## 合规检查清单

- [ ] 是否维护了完整的材料数据库（IMDS或等效）？
- [ ] 所有供应商是否提供了四种有害物质的合规声明？
- [ ] 含铅/六价铬的应用是否对应有效的Annex II豁免？
- [ ] 豁免到期日是否监控并有替代计划？
- [ ] 是否按均质材料（非零部件整体）计算含量？
- [ ] Annex II要求标识"含Pb"的部件是否已标识？
- [ ] 是否启用了XRF/ICP-OES等现场快速筛查？
- [ ] 跟踪了Annex II的历次修订（最新2023版）？
- [ ] 新ELV法规提案的影响是否评估？

## 常见不合规情况

1. **中国低端供应商六价铬残留**：宣称三价铬钝化但测试发现Cr(VI)残留（常见于螺丝、弹簧等紧固件）。
2. **铅合金钢超标**：历史供应商使用含铅易切削钢（铅含量>0.35%），违反1(a)豁免限值。
3. **NiCd电池在早期HEV车型中残留**：2005-2010前的HEV设计可能仍用NiCd，进入EU前需替换。
4. **豁免到期未监控**：如2(c)(ii)轴承铅将于2026-07-01到期，但OEM未跟踪。
5. **标识遗漏**：含Pb焊料未在PCB上打"Pb"标识，符合含量但违反标识要求。
6. **涂装CrVI钝化库存未清**：改制Cr+3后原库存板材/紧固件仍有老工艺。
7. **整车合规但备件不合规**：售后备件供应商不熟悉ELV，仍用老工艺供货。

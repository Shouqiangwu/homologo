# ELV Annex II 豁免清单（完整对照）

> ELV Directive 2000/53/EC Annex II（2023最新修订版）

---

## 豁免清单的性质

ELV Art.4(2)(a)原则上禁止在汽车中使用铅、汞、镉、六价铬。Annex II列出允许的例外，每个豁免条目指明：
- 允许的具体应用
- 最大含量限值
- 到期日（部分有，部分无）
- 标识要求（部分豁免要求零部件上打标）

**注意：** Annex II由欧委会定期审查修订，历史上已修订**14次**。部分豁免已废止，部分新增——必须查阅最新版本。

## 2023修订重点

| 修订 | 内容 |
|------|------|
| 滑动轴承铅（2(c)(ii)） | 到期日从2024延至2026-07-01 |
| 压电陶瓷（条目7） | 应用范围收紧 |
| 焊料（条目8系列） | 部分应用的高铅焊料延期 |
| 新增审查 | 欧委会启动2025-2027新一轮Annex II评审 |

## 与REACH的协调

| 物质 | ELV Annex II | REACH位置 | 优先适用 |
|------|-------------|----------|---------|
| 铅 | 有豁免 | SVHC候选、Annex XVII Entry 63 | 车辆部件以ELV为准；消费件（方向盘皮革等）以Annex XVII为准 |
| Cr(VI) | 有部分豁免 | SVHC候选、Annex XIV | 已过渡到Cr+3，授权快到期 |
| 镉 | 有限豁免 | Annex XVII Entry 23 | Annex XVII更严（0.01%），以更严为准 |
| 汞 | 基本无豁免 | 全球已逐步禁用 | 全禁 |

## 合规检查清单

- [ ] 车辆所有使用铅的部件是否有对应Annex II豁免条目？
- [ ] 豁免到期日是否在产品生命周期内？
- [ ] 豁免要求的标识是否已执行（如"含Pb"标签）？
- [ ] 是否跟踪了Annex II历次修订？
- [ ] Cr(VI)钝化是否已全面替代为Cr+3？
- [ ] NiCd/汞灯/含镉接触器是否已完全淘汰？
- [ ] 供应商合规声明是否引用具体豁免条目号？

## 常见不合规情况

1. **"铜合金含铅"豁免错用**：2(c)(i)是车辆一般铜件，2(c)(ii)是轴承衬套特定应用——常被混用。
2. **到期豁免未切换**：4(a)/(b)减震铅早已废止，但老车型零部件库存仍在使用。
3. **标识缺失**：豁免5/8(f)/8(h)要求"含Pb"标识，但多数供应商不标。
4. **2023修订未同步**：按2018版Annex II核查，忽略2023年的延期和收紧。
5. **Cr(VI)钝化残留**：部分低端供应商宣称已改Cr+3但实际仍用Cr(VI)——需批次检测。

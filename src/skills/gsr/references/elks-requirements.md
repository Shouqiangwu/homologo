# ELKS 紧急车道保持系统要求

> UNECE R130 — Emergency Lane Keeping System

---

## 适用范围
- **适用类别：** M1/N1（液压助力转向的M2/M3/N2/N3按R130可选择LDWS替代）
- **新车型强制日期：** 2022-07-06
- **所有新车强制日期：** 2024-07-07

## 系统定义
ELKS是**主动纠正系统**，当车辆即将无意越过车道标线时，自动施加转向力矩将车辆引回车道内。这与LDW（车道偏离警告，仅警告不纠正）和LKA（车道保持辅助，持续居中）有本质区别。

**关键区分：**
- LDW（车道偏离警告）= 仅警告 → **不满足**GSR要求
- ELKS（紧急车道保持）= 检测+主动纠正 → **满足**GSR要求
- LKA（车道保持辅助/车道居中）= 持续辅助 → 超出要求，但仍需满足ELKS基本要求

## 核心技术要求 — R130

| 要求项 | 具体阈值/要求 | 法规条款 |
|--------|-------------|---------|
| 激活速度范围 | ≥70 km/h（部分版本≥60 km/h） | §5.1 |
| 车道标线识别 | 必须能识别实线和虚线车道标线 | §5.2 |
| 纠正动作 | 施加转向力矩，将车辆引回车道内 | §5.3 |
| 驾驶员可覆盖 | 驾驶员通过转向操作可随时覆盖ELKS | §5.4 |
| 警告 | 纠正动作启动时需提供至少1种警告（声觉/触觉/视觉） | §5.5 |
| 转向灯抑制 | 当驾驶员打转向灯时ELKS不得干预 | §5.6 |
| 每次启动默认激活 | 是 | §5.7 |
| 可关闭 | 驾驶员可手动关闭，点火重启后自动恢复 | §5.8 |

## 合规检查清单

- [ ] 系统是否为ELKS（主动纠正）而非仅LDW（仅警告）？
- [ ] 系统是否在≥70 km/h时激活？
- [ ] 是否能识别实线和虚线车道标线？
- [ ] 驾驶员是否可通过转向操作覆盖？
- [ ] 打转向灯时系统是否不干预？
- [ ] 每次启动后是否默认激活？
- [ ] 关闭后点火重启是否自动恢复？

## 常见不合规情况

1. **系统仅有LDW功能**：中国部分车型仅配备车道偏离警告（LDW），不含主动纠正功能。需升级为ELKS。
2. **低速激活**：部分系统在城市道路低速（<60km/h）即激活ELKS，可能导致误触发。R130要求≥70km/h。
3. **转向灯抑制逻辑**：打转向灯后ELKS延迟抑制时间过短或过长，影响正常变道。

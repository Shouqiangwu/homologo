# 供应链尽职调查要求

> EU 2023/1542 Art.48-52（修正：EU 2025/1561延期至2027-08-18）

---

## 适用范围

- **适用对象：** 将电池投放EU市场的经济运营商（制造商/进口商），年净营业额>€4000万（修正后阈值，原为€150M）
- **适用电池类别：** EV、工业、LMT
- **强制日期：** 2027-08-18（原定2025-08-18，通过EU 2025/1561延期2年）
- **覆盖原材料：** 钴、天然石墨、锂、镍

## 核心义务

| 义务 | 具体要求 | Art. |
|------|---------|------|
| 尽职调查政策 | 制定并公开发布供应链尽职调查政策 | Art.48 |
| 管理体系 | 建立供应链管理体系，指定高级管理人员负责 | Art.48 |
| 供应商管控 | 评估供应商的社会和环保风险 | Art.49 |
| 风险识别和缓解 | 识别供应链中的人权和环保风险并采取缓解措施 | Art.49 |
| 申诉机制 | 建立有效的申诉渠道 | Art.50 |
| 第三方验证 | 由认证机构（notified body）进行第三方验证 | Art.51 |
| 公开报告 | 定期公开发布尽职调查报告 | Art.52 |

## 风险关注领域

| 风险类别 | 示例 |
|---------|------|
| 人权风险 | 刚果（金）钴矿童工、强制劳动 |
| 环境风险 | 锂盐湖开采的水资源消耗、镍矿冶炼的污染排放 |
| 社区影响 | 矿区社区搬迁、生计破坏 |
| 治理风险 | 冲突矿产、腐败、非法采矿 |

## 合规检查清单

- [ ] 是否制定了供应链尽职调查政策并公开发布？
- [ ] 是否指定了高级管理人员负责尽职调查？
- [ ] 是否对钴、石墨、锂、镍的供应链进行了风险评估？
- [ ] 是否有供应商社会和环保风险的评估记录？
- [ ] 是否建立了申诉渠道？
- [ ] 是否安排了第三方验证（notified body）？
- [ ] 是否定期公开发布尽职调查报告？

## 常见不合规情况

1. **供应链可见度不足**：大多数中国电池企业对二级以上供应商（矿端）缺乏可见度。
2. **刚果钴矿溯源挑战**：全球约70%的钴产自刚果（金），人权风险集中。
3. **尽职调查报告未公开**：中国企业不习惯公开发布供应链风险报告。

# 酒精锁预留接口要求

> EU 2021/1243 — 补充EU 2019/2144

---

## 适用范围

- **适用类别：** 全部M/N类
- **新车型强制日期：** 2022-07-06
- **所有新车强制日期：** 2024-07-07
- **状态：** ✅ 已完全强制

## 要求说明

**注意：** GSR**不要求**在车辆中安装酒精锁（Alcohol Interlock Device），而是要求预留**标准化的安装接口**（facilitation of the installation）。

### 核心要求 — EU 2021/1243

| 要求项 | 具体要求 |
|--------|---------|
| 预留接口 | 车辆必须预留一个标准化接口，允许日后安装酒精锁设备 |
| 接口标准 | 符合EN 50436系列标准 |
| 接口位置 | 驾驶员可触及范围内，便于使用 |
| 电气接口 | 提供电源连接和与车辆启动系统的信号接口 |
| 不影响正常使用 | 预留接口不得影响车辆正常功能 |

### 接口技术规格

| 参数 | 要求 |
|------|------|
| 电源 | 提供12V/24V电源接口 |
| 通信 | 与车辆启动锁止系统的信号接口 |
| 物理空间 | 预留安装酒精锁设备的物理空间 |
| 标准兼容 | 与EN 50436-1/2/3/4/5/7系列标准兼容 |

## 合规检查清单

- [ ] 车辆是否预留了酒精锁安装接口？
- [ ] 接口是否符合EN 50436系列标准？
- [ ] 接口是否位于驾驶员可触及范围内？
- [ ] 是否提供了电源和信号接口？
- [ ] 预留接口是否不影响车辆正常功能？
- [ ] 技术文档中是否有酒精锁接口的描述和图纸？

## 常见不合规情况

1. **完全未考虑酒精锁接口**：这是中国车企最容易忽略的GSR要求之一，因为中国法规无此要求。
2. **接口标准不兼容**：使用了非EN 50436系列的私有接口。
3. **技术文档缺失**：型式认证文件中未包含酒精锁接口的描述。

# 空调系统泄漏率测试

> ECE R82 + SAE J2763 + MAC Directive Annex III

---

## 泄漏率法规要求概述

| 系统类型 | 最大年泄漏率 | 适用类别 |
|---------|------------|---------|
| 单蒸发器A/C | 40 g/年 | M1, N1 Class I |
| 双蒸发器A/C（如MPV后舱独立空调） | 60 g/年 | M1, N1 Class I |

超过限值→型式认证不通过。

## 测试程序详解（ECE R82 / SAE J2763）

### 测试箱要求

| 参数 | 要求 |
|------|------|
| 密闭性 | 可精确测量制冷剂质量变化 |
| 温度控制 | 40°C ± 5°C（一些方案允许35°C或45°C分级测试） |
| 湿度 | 控制避免冷凝影响称重 |
| 气流 | 避免强制对流干扰 |
| 测试空间 | 足以容纳整车或A/C系统模拟台架 |

### 测试方法1：整车称重法

```
Step 1: 车辆进入测试箱，预处理（温度稳定）至40°C - 4小时
Step 2: 精密称重仪记录初始质量 m0
Step 3: 发动机启动，A/C最大制冷运转 - 5分钟
Step 4: 发动机关闭，车辆静置
Step 5: 24小时后再次称重 m24
Step 6: 泄漏量 = m0 - m24（g）
Step 7: 年化：24h × 365 × 外推因子（通常1.0）
```

### 测试方法2：示踪气体法（SF6）

```
Step 1: A/C系统充注制冷剂 + 0.1% SF6示踪气体
Step 2: 在测试箱内运转并静置
Step 3: 测试箱内气体采样 → GC-ECD检测SF6浓度
Step 4: 根据SF6释放速率计算制冷剂泄漏速率
```

适用于极小泄漏量测试（<10 g/年），称重法精度不足时使用。

### 测试方法3：组件级泄漏测试

| 组件 | 典型测试方法 | 允许泄漏 |
|------|------------|---------|
| 压缩机轴封 | 氦质谱检漏 | ≤5 g/年 |
| 软管接头 | 压力衰减 + 氦检漏 | ≤2 g/年/接头 |
| O-rings | 加速老化 + 氦检漏 | ≤1 g/年/接头 |
| 冷凝器/蒸发器 | 氦质谱或氮气压力测试 | ≤1 g/年 |
| 膨胀阀 | 氦质谱 | ≤2 g/年 |

整车泄漏率 = 所有组件泄漏率之和 + 系统整体效应

## 关键泄漏点分析

### 压缩机轴封（典型占比30-40%）

| 问题 | 影响 |
|------|------|
| 动态密封面磨损 | 随运行时间增加 |
| 冷启动泄漏 | 冬季油封收缩 |
| 电动压缩机特殊 | BEV使用电动压缩机，全密封设计泄漏更低 |

**BEV优势：** 电动压缩机的全封闭设计（不需旋转轴穿出壳体）可将压缩机泄漏降至≤2 g/年。

### 软管和接头（典型占比20-30%）

| 类型 | 泄漏风险 |
|------|---------|
| 高低压软管 | 橡胶随温度老化，EPDM > NBR 的R-1234yf兼容性 |
| 快接接头 | O-ring密封，设计和装配敏感 |
| Schrader阀（维修口） | 高风险点，帽盖需密封二次保障 |

### 蒸发器（典型占比10-20%）

| 问题 | 影响 |
|------|------|
| 腐蚀 | 乘员舱冷凝水+空气污染 |
| 振动疲劳 | 焊接点 |
| 异物刺穿 | 极少但严重 |

### 冷凝器

| 问题 | 影响 |
|------|------|
| 石击损伤 | 发动机舱前部暴露 |
| 盐蚀 | 沿海和冬季道路撒盐 |

## 泄漏率设计目标

### OEM内部设计指标（比法规严）

| 设计阶段 | 年泄漏率目标 |
|---------|------------|
| 量产初期 | ≤25 g/年（单蒸发器） |
| 设计寿命末期（10-15年） | ≤40 g/年（不超限值） |
| 考虑公差 | 设计值25 g/年，3σ公差至35 g/年 |

### 长期可靠性（LTR）测试

除型式认证测试外，OEM应执行加速老化测试：
- 热循环：-40°C至+85°C，1000次循环
- 振动测试：模拟100,000 km使用
- 加速老化后再测泄漏率

## 测试常见错误

### 实验室错误

1. **温度不稳定**：测试箱温度波动±8°C导致制冷剂液汽相态变化影响称重
2. **称重漂移**：24h测试中秤漂0.5g，叠加在泄漏测量上
3. **预处理不足**：4小时预处理未达内部平衡，系统仍在释放气态R-1234yf
4. **环境干扰**：测试箱开门取样导致温度重建

### 合规策略错误

1. **仅做一次测试**：单次40 g/年"过"，但实际生产离散性大
2. **不做组件级追踪**：总泄漏率临界后无法定位根因
3. **忽略高温应力**：测试40°C，但车辆实际经历的暴晒发动机舱>80°C
4. **R-134a与R-1234yf混用经验**：R-1234yf分子大小略小、密封材料兼容性不同

## 与长期合规的关系

MAC Directive测试是型式认证初始状态，但：

| 问题 | 影响 |
|------|------|
| 无强制"现场泄漏率"测量 | 老化问题不直接罚款 |
| 但F-GAS法规仍管 | 若年大批车辆泄漏严重，可能触发市场监督调查 |
| ISC关联 | 在用车排放数据可能间接反映A/C泄漏 |

## 合规检查清单

- [ ] A/C系统是否在ECE R82测试中通过40/60 g/年限值？
- [ ] 测试是否由EU认可Technical Service执行？
- [ ] 设计泄漏率是否有余量（<25 g/年为目标）？
- [ ] 是否有组件级泄漏率分解数据？
- [ ] 是否做了长期可靠性测试（加速老化）？
- [ ] 量产品质管控中是否有泄漏率抽检？
- [ ] 售后维修后是否有泄漏检测程序？
- [ ] 关键泄漏点（压缩机轴封、接头）是否可追溯供应商？

## 常见不合规情况

1. **BEV直接套用ICE设计**：BEV使用电动压缩机但管路设计沿用ICE，浪费密封优势。
2. **R-1234yf密封件未升级**：原R-134a系统直接切换，O-ring和软管未改为兼容R-1234yf的HNBR/EPDM。
3. **SUV双蒸发器未加强密封**：双蒸发器限值60g/年看起来宽松，但实际多一套软管和接头，泄漏叠加。
4. **泄漏率测试样本量不足**：仅1辆样车测试"过了"，未评估生产离散性。
5. **维修后未检测**：换压缩机/蒸发器后未做泄漏率复测。
6. **高温地区泄漏异常**：南欧、中东市场车辆泄漏率因气温高偏高。

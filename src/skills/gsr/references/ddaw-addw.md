# DDAW + ADDW 驾驶员状态监测系统要求

> EU 2021/1341（DDAW）+ EU 2023/2590（ADDW）

---

## 两个系统的区分

| | DDAW 困倦/注意力警告 | ADDW 高级分心警告 |
|--|---------------------|-------------------|
| **全称** | Driver Drowsiness and Attention Warning | Advanced Driver Distraction Warning |
| **法规** | EU 2021/1341 | EU 2023/2590 |
| **功能** | 检测驾驶员困倦/注意力下降 | 检测驾驶员视觉分心（目光偏离道路） |
| **技术手段** | 可基于车辆动态数据（方向盘输入模式等）或驾驶员监测 | **必须**监测驾驶员目光方向（需驾驶员监测摄像头DMS） |
| **新车型强制** | 2022-07-06 | 2024-07-07 |
| **所有新车强制** | 2024-07-07 | 2026-07-07 |
| **适用类别** | 全部M/N | 全部M/N |

**关键区别：** DDAW可以不用摄像头（基于车辆行为推断），ADDW必须用驾驶员监测摄像头。

## DDAW 技术要求 — EU 2021/1341

| 要求项 | 具体要求 | 条款 |
|--------|---------|------|
| 检测方法 | 基于车辆行为分析（方向盘角度/频率等）和/或驾驶员直接监测 | Art.3 |
| 激活速度 | ≥70 km/h（对于基于车辆行为的系统） | Art.4 |
| 警告方式 | 至少视觉警告，可附加声觉/触觉 | Art.5 |
| 警告级联 | 初始警告→如驾驶员未响应→增强警告 | Art.5 |
| 默认激活 | 每次启动后自动激活 | Art.6 |
| 可关闭 | 是，但重启后恢复 | Art.6 |
| 数据记录限制 | 不得持续记录/保留驾驶行为数据；闭环处理后立即删除 | Art.7 |

## ADDW 技术要求 — EU 2023/2590

| 要求项 | 具体要求 | 条款 |
|--------|---------|------|
| 检测方法 | **必须**监测驾驶员目光方向（gaze direction） | Art.3(1) |
| 传感器 | 需要驾驶员监测摄像头（DMS Camera），通常位于转向柱或仪表板上方 | Art.3(2) |
| 激活条件 | 车辆行驶中，驾驶员应注视道路前方 | Art.4 |
| 分心判定 | 驾驶员目光持续偏离道路前方超过阈值时间 | Art.5 |
| 警告方式 | 视觉+至少一种其他模式（声觉/触觉） | Art.6 |
| 警告级联 | Level 1（提醒）→ Level 2（强化警告）→ Level 3（紧急措施建议） | Art.6 |
| 与DDAW协调 | ADDW和DDAW不得同时触发，不得以混淆方式分别/同时提示驾驶员 | Art.7 |
| 默认激活 | 每次启动后自动激活 | Art.8 |
| 数据记录限制 | 同DDAW，不得持续记录，闭环处理后立即删除 | Art.9 |

### ADDW 分心判定阈值（EU 2023/2590 Annex）

| 分心类型 | 判定条件 | 警告触发 |
|---------|---------|---------|
| 持续视觉分心 | 目光持续偏离道路前方 | 连续偏离时间超过系统设定的阈值（由OEM标定） |
| 频繁短暂分心 | 在一定时间窗口内频繁短暂偏离 | 累积偏离频率超过阈值 |

**注意：** EU 2023/2590的测试程序允许一定的不确定性，因为ADDW需要解读人类行为。测试中如果出现不通过，可在调整分心行为细节后重测。

## 合规检查清单

### DDAW
- [ ] 系统是否安装并默认激活？
- [ ] 检测方法是什么？（车辆行为分析/驾驶员直接监测/两者结合）
- [ ] 是否在≥70km/h时激活（基于车辆行为的系统）？
- [ ] 警告是否包含视觉模式？
- [ ] 是否有级联警告机制？
- [ ] 是否不持续记录驾驶行为数据？

### ADDW
- [ ] 是否配备驾驶员监测摄像头（DMS）？
- [ ] DMS是否能检测驾驶员目光方向？
- [ ] 分心判定逻辑是否定义（持续分心阈值、频繁分心阈值）？
- [ ] 警告是否包含视觉+至少一种其他模式？
- [ ] 是否有Level 1→2→3级联机制？
- [ ] ADDW和DDAW是否不会同时冲突触发？
- [ ] 是否不持续记录驾驶员面部图像/目光数据？
- [ ] DMS数据是否闭环处理后立即删除？

## 常见不合规情况

1. **无DMS摄像头的车型不满足ADDW**：2024-07-07起新车型必须有ADDW，即必须有DMS。部分中低端车型可能未配备。
2. **DMS数据上传到云端**：部分中国车企的DMS数据用于远程诊断或AI训练，违反数据记录限制。
3. **ADDW和DDAW警告冲突**：两个系统同时触发导致驾驶员困惑（EU 2019/2144 Art.7(3)明确禁止此情况）。
4. **ADDW仅有视觉警告**：EU 2023/2590要求视觉+至少一种其他模式，仅有仪表盘图标不够。
5. **DMS因佩戴墨镜/口罩失效**：系统需在常见遮挡条件下仍能工作或明确提示无法工作。

# 碳足迹声明和限值要求

> EU 2023/1542 Art.7 + Annex II

---

## 三阶段递进框架

EU电池法规对碳足迹采用三阶段递进式管理：

| 阶段 | 内容 | EV电池时间 | 说明 |
|------|------|-----------|------|
| **阶段1：声明** | 计算并声明电池全生命周期碳足迹 | 2025-02-18起 ✅已强制 | 每个电池型号/每个工厂单独计算 |
| **阶段2：等级标签** | 按碳足迹值分配性能等级（如A-G），标签展示 | 2026-08-18起 | 委托法规定义等级划分标准 |
| **阶段3：最高限值** | 超过限值的电池不得在EU市场销售 | 2027-07起（委托法规发布） | 最严格阶段，市场准入门槛 |

## 碳足迹计算方法 — Art.7 + Annex II

### 计算范围（全生命周期）

碳足迹必须覆盖以下阶段的CO₂当量排放：

| 生命周期阶段 | 包含内容 |
|-------------|---------|
| 原材料获取 | 锂、钴、镍、石墨等矿物开采和加工 |
| 前驱体和正极材料生产 | 化学合成过程 |
| 电芯制造 | 涂布、组装、化成、分容 |
| 电池包组装 | BMS、热管理系统、壳体等 |
| 运输 | 原材料→工厂→市场的物流 |
| 使用阶段 | 充放电过程中的电力来源碳强度（部分要求） |
| 回收处理 | 回收过程的能耗和回收材料的碳信用 |

### 功能单位

| 电池类别 | 功能单位 |
|---------|---------|
| EV电池 | kg CO₂e / kWh（总储能量） |
| 工业电池 | kg CO₂e / kWh |
| LMT电池 | kg CO₂e / kWh |

### 声明格式要求

| 要求项 | 具体要求 |
|--------|---------|
| 计算方法 | 必须使用EU委托法规规定的统一方法学（基于PEF方法） |
| 数据质量 | 优先使用实际数据（primary data），缺失时可用通用数据库 |
| 第三方验证 | 需经认证机构（notified body）验证 |
| 公开披露 | 碳足迹值通过技术文档和电池护照可查 |
| 工厂粒度 | 每个制造工厂单独计算（不同工厂同型号电池碳足迹可能不同） |

## 合规检查清单

- [ ] 是否已计算EV电池的全生命周期碳足迹？（2025-02-18起强制）
- [ ] 计算方法是否符合EU委托法规规定的统一方法学？
- [ ] 碳足迹功能单位是否为kg CO₂e / kWh？
- [ ] 计算范围是否覆盖全生命周期（原材料→制造→运输→使用→回收）？
- [ ] 数据来源是否可追溯（primary data vs 数据库数据比例）？
- [ ] 是否针对每个制造工厂分别计算？
- [ ] 碳足迹声明是否经第三方验证？
- [ ] 碳足迹值是否纳入技术文档和CE符合性评估？
- [ ] 是否准备了碳足迹性能等级标签？（2026-08-18起需要）

## 常见不合规情况

1. **碳足迹计算方法不符合EU规定**：使用了中国国标方法（如GB/T 24067）或ISO 14067，但未按EU PEF方法学重新计算。EU要求使用其特定的方法学框架。
2. **供应链数据缺失**：上游原材料（锂矿、钴矿）的碳排放数据缺乏，使用了通用数据库中的全球平均值而非实际供应商数据。这在阶段1可接受但在阶段3可能不足。
3. **未按工厂粒度计算**：同一型号电池在不同工厂生产（如中国工厂vs匈牙利工厂），碳足迹不同但未分别计算。
4. **运输碳排放被低估**：从中国运输至EU的海运碳排放未充分计入。
5. **电力碳强度问题**：中国电网碳强度显著高于EU平均水平。如果电芯在中国生产，碳足迹值可能远高于EU本地生产的竞品。这是中国电池企业面临的结构性挑战。

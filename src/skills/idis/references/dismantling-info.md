# 拆解信息提交（IDIS平台）

> ELV Directive 2000/53/EC Art.8 + IDIS 2 平台规范

---

## IDIS 2 平台概述

IDIS（International Dismantling Information System）是汽车行业为满足ELV Art.8拆解信息要求建立的共享平台。

| 项目 | 详情 |
|------|------|
| 平台域名 | www.idis2.com |
| 运营方 | IDIS Consortium（多家OEM联合） |
| 成员OEM | 包括Stellantis、Ford、Toyota、VW等60+ OEM |
| 使用方 | ATF（授权处理设施）、回收企业、拆解技师 |
| 数据覆盖 | >3,500车型，跨40+年 |
| 语言 | 英文为主，部分数据有本地语言版本 |

## 法规基础

ELV Directive 2000/53/EC Art.8(3):
> 制造商须在新车型投放市场后6个月内，以可自由获取的方式向处理设施提供车辆拆解信息。

**IDIS是行业认可的满足Art.8的"默认方式"**，但不是法规强制唯一方式。部分OEM也提供自有网站平台。

## 提交内容的IDIS标准模板

### 车辆基础信息

| 字段 | 数据类型 | 示例 |
|------|---------|------|
| Make | 文本 | 比亚迪 |
| Model | 文本 | Atto 3 |
| Variants | 文本 | EV Standard Range / Extended Range |
| Production years | 时间段 | 2022-present |
| Type approval number | 编号 | e1*2018/858*00046*01 |
| Category | 分类 | M1 |
| Curb weight | 质量 | 1,750 kg |
| Drivetrain | 分类 | BEV / PHEV / HEV / ICE |

### 预处理信息（Pre-treatment）

| 项目 | 子字段 |
|------|--------|
| 燃油 | 位置、容量、排放点 |
| 机油 | 位置、容量、排放点 |
| 冷却液 | 位置、容量、排放点（含BEV电池冷却液） |
| 制动液 | 位置、容量、排放点 |
| 转向液 | 位置、容量（BEV通常无液压动力） |
| 空调制冷剂 | 类型（R-1234yf等）、充注量、维修端口位置 |
| 窗洗液 | 位置、容量 |
| 差速器/变速箱油 | 位置、容量 |
| AdBlue（柴油） | 位置、容量 |

### 爆炸装置信息

| 项目 | 要求 |
|------|------|
| 安全气囊 | 所有气囊位置图示 + 中和步骤 |
| 安全带预紧器 | 位置 + 中和 |
| 车顶气囊 | 位置 |
| 外部行人保护气囊 | 位置（如有） |

### 电池信息

| 电池类型 | 提交内容 |
|---------|---------|
| **启动电池** | 位置、类型（铅酸/锂离子）、容量（Ah）、拆卸步骤 |
| **EV高压电池** | 详见 bev-battery-dismantling.md |
| **12V辅助电池（EV）** | 同启动电池 |
| **记忆电池/ECU电池** | 位置（内置电池可能含有害物质） |

### 催化转化器

| 信息 | 说明 |
|------|------|
| 位置 | 排气系统示意图 |
| 贵金属含量 | Pt/Pd/Rh含量（有价回收用） |
| 拆除步骤 | 避免损坏 |

### 主要零部件位置图

- 仪表板拆卸步骤（气囊中和后）
- 座椅、座椅加热装置、座椅安全带
- 车门内饰
- 保险杠（含雷达/摄像头ADAS传感器）
- 动力总成（发动机/电机）
- 变速箱/减速器
- 悬挂系统

### 材料标识

按ISO 1043（塑料）和ISO 11469（零件）要求的材料标识位置：

| 常见塑料 | ISO 1043代号 |
|---------|-------------|
| 聚丙烯 | PP |
| 聚丙烯+滑石 | PP-T20 |
| 聚乙烯 | PE |
| 聚氨酯 | PUR |
| ABS | ABS |
| PC/ABS | PC+ABS |

### 有害物质位置（Annex II豁免项目）

| 物质 | 典型位置 |
|------|---------|
| 铅（合金钢） | 易切削零件 |
| 铅（铜合金轴承） | 悬架衬套 |
| 铅（焊料） | PCB板上的焊点 |
| 铅（压电陶瓷） | 某些传感器 |
| 汞（如有老车型） | HID氙灯 |
| 镉（NiCd老车型） | HEV电池（已基本淘汰） |

## 提交流程

### 步骤1：OEM注册

```
访问 www.idis2.com
    ↓
申请OEM账号（通常需€数千到数万/年订阅费）
    ↓
签署IDIS数据提供者协议
    ↓
获得数据上传权限
```

### 步骤2：数据准备

```
OEM内部拆解工程团队汇编信息
    ↓
按IDIS标准模板整理（XML格式 + 图示附件）
    ↓
内部质量审核
    ↓
翻译为英文（可选本地语言版本）
```

### 步骤3：上传和审核

```
通过IDIS portal上传数据包
    ↓
IDIS审核员初步检查完整性
    ↓
OEM修正（如有缺失）
    ↓
发布到ATF可见
```

### 步骤4：维护更新

| 触发 | 更新 |
|------|------|
| 车型改款 | 相关部分更新 |
| 软件版本变化（影响拆解） | 更新 |
| 法规变化 | 如新Annex II豁免 |
| ATF反馈 | 基于实际拆解经验的改进 |
| 停产后 | 维护≥15年 |

## 数据质量要求

### 完整性

IDIS审核流程会检查必填字段，缺失字段导致上传失败。

### 准确性

- 拆卸步骤与实车一致（非设计早期版）
- 扭矩/尺寸准确
- 图示清晰可辨

### 可用性

- 步骤逻辑顺序
- 重要警告突出
- 专业术语与通俗表达并存

### 时效性

- 上市后6个月内完成初始提交
- 变更后3个月内更新

## 中国车企的常见问题

### 问题1：不知道IDIS存在

中国车企首次进EU市场常不了解IDIS，误以为只需向某国家机构提交拆解信息。

### 问题2：订阅费预算

IDIS订阅费（OEM数据提供方€数千到数万/年）常未在首次EU市场进入预算中考虑。

### 问题3：数据格式

中国内部维修手册是PDF/Word格式，而IDIS要求结构化XML——需专门转换工作。

### 问题4：英文翻译质量

拆解技术英文术语专业性强，简单机器翻译常有错误。

### 问题5：EV数据复杂度

BEV的高压系统、电池包信息比ICE复杂很多（见 bev-battery-dismantling.md）。

## IDIS替代方案

如OEM不使用IDIS，需自建平台：

| 要求 | 说明 |
|------|------|
| 可自由获取 | ATF无需付费即可获得 |
| 英文+本地语言 | 目标市场语言 |
| 结构化数据 | 便于快速查找 |
| 与IDIS等效 | 覆盖所有IDIS必填内容 |

**注意：** 自建平台初期成本高，维护成本也高——中小OEM使用IDIS更经济。

## 合规检查清单

- [ ] 是否在IDIS平台注册了OEM账号？
- [ ] 新车型是否在上市6个月内完成了初始提交？
- [ ] 预处理信息（所有油液、气囊）是否完整？
- [ ] EV高压电池信息是否包含？
- [ ] 安全气囊/预紧器中和步骤是否清楚？
- [ ] 主要零部件位置图示是否清晰？
- [ ] 材料标识位置是否按ISO 1043/11469说明？
- [ ] 有害物质位置（Annex II豁免）是否标注？
- [ ] 空调制冷剂类型和充注量是否记录？
- [ ] 是否有英文版本？
- [ ] 车型改款时是否及时更新？
- [ ] 停产后是否持续维护≥15年？

## 常见不合规情况

1. **超6个月提交**：上市8-12个月后才开始IDIS数据工作。
2. **仅中文不合规**：未提供英文版本。
3. **EV信息简陋**：高压安全和电池拆卸步骤不足。
4. **图示质量差**：用CAD线图，无实物照片。
5. **扭矩/尺寸错误**：引用设计值而非量产值。
6. **材料标识位置描述模糊**："发动机舱内"等不精确。
7. **改款未更新**：2022款和2023款差异大但IDIS仅有2022版。
8. **停产后数据消失**：OEM取消IDIS订阅，ATF无法访问老车数据。

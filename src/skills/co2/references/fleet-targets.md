# 车队CO2目标和罚款机制

> EU 2019/631 + EU 2023/851

---

## EU车队平均CO2排放目标（M1乘用车）

| 时间段 | 车队平均目标 | 相对2021基线 | 说明 |
|--------|------------|-------------|------|
| 2021-2024 | ~115.1 g/km (WLTP) | 基线 | 等效于95 g/km NEDC |
| **2025-2029** | **93.6 g/km (WLTP)** | -15% | ✅ 已生效 |
| **2030-2034** | **49.5 g/km (WLTP)** | -55% | 即将生效 |
| **2035起** | **0 g/km** | -100% | 仅零排放车辆可注册 |

**N1轻型商用车：** 2025目标153.9 g/km；2030目标-50%；2035目标0 g/km。

## 2025-2027合规灵活性（2025年修正提案）

欧委会2025年提出修正案（待通过），允许2025-2027年按**三年平均**（banking & borrowing）计算合规，而非逐年：
- 如某年超标但三年平均达标，不罚款
- 该灵活性仅限2025-2027，2028起恢复逐年合规

## 超标罚款机制

| 参数 | 数值 |
|------|------|
| 罚款单价 | **€95 / 每超标1 g/km** |
| 计算方式 | 罚款 = €95 × 超标克数 × 该年EU注册量 |
| 示例 | 某车企车队平均超标5 g/km，年注册10,000辆 → 罚款=€95×5×10,000 = **€4,750,000** |

## 制造商特定目标计算

每个制造商的具体目标不是统一的93.6 g/km，而是根据其车队平均**整备质量**（mass in running order）调整：
- 车队平均质量越重 → 允许的CO2目标越高
- 车队平均质量越轻 → 允许的CO2目标越低
- 计算公式见EU 2019/631 Annex I Part A Point 6

## 小批量制造商豁免

| 年注册量 | 豁免条件 |
|---------|---------|
| <1,000辆 | 不受CO2标准约束 |
| 1,000-10,000辆 | 可申请豁免目标（derogation），至2035年 |
| 10,000-300,000辆 | 可申请豁免目标，至2028年 |

## 合规检查清单

- [ ] 车企在EU的年注册量是多少？是否触发CO2标准？
- [ ] 车队平均CO2排放（WLTP）是否≤制造商特定目标？
- [ ] 如为BEV/FCEV，是否正确计为0g/km？
- [ ] 如为PHEV，WLTP CO2值是否正确计算（含Utility Factor）？
- [ ] 是否考虑了pooling策略？
- [ ] 是否了解超标罚款的财务风险？
- [ ] 年度监控数据报告是否按EU 2021/392执行？

## 常见不合规情况

1. **未加入pool**：中国小品牌独立进入EU时注册量小但ICE/PHEV车型CO2偏高，未加入pool导致超标罚款。
2. **PHEV实际排放偏高**：PHEV的WLTP CO2取决于Utility Factor和电续航里程，实际使用中燃油模式占比高会在ISC中暴露。
3. **未做年度报告**：新进入EU市场的车企不熟悉EU 2021/392的年度监控报告流程。

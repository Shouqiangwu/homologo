# 可回收率计算

> ISO 22628 + EU 2005/64/EC

---

## 计算方法 — ISO 22628

| 概念 | 定义 | 目标 |
|------|------|------|
| **可回收率（Recyclability Rate）** | 可通过材料回收方式处理的车辆质量占比 | ≥85% |
| **可利用率（Recoverability Rate）** | 可通过材料回收+能量回收方式处理的车辆质量占比 | ≥95% |

### 计算公式

```
可回收率 = (可重用质量 + 可回收质量) / 整车质量 × 100%
可利用率 = (可重用质量 + 可回收质量 + 可能量回收质量) / 整车质量 × 100%
```

### 质量分类

| 分类 | 说明 | 示例 |
|------|------|------|
| 可重用 | 可直接再使用的零部件 | 发动机、变速箱（翻新后） |
| 可回收 | 可通过材料回收处理 | 金属件、塑料件（已标识材料） |
| 可能量回收 | 可通过燃烧回收能量 | 部分塑料、橡胶 |
| 不可利用 | 既不可回收也不可能量回收 | 部分复合材料、粉尘 |

## 型式认证中的提交要求

- 计算报告需在型式认证阶段提交
- 由Technical Service审核
- 需覆盖整车所有主要材料和零部件

## 合规检查清单

- [ ] 是否按ISO 22628完成了可回收率/可利用率计算？
- [ ] 可回收率是否≥85%？
- [ ] 可利用率是否≥95%？
- [ ] 计算是否覆盖了EV电池质量？
- [ ] 计算报告是否纳入型式认证文件？

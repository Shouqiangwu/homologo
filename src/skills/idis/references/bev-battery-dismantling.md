# BEV电池拆解专项信息

> IDIS规范 + Battery Reg 2023/1542 + UNECE R100/R136

---

## BEV电池拆解的法规背景

```
ELV Directive 2000/53/EC Art.8
└─ 要求拆解信息

Battery Reg 2023/1542 Art.11
└─ 要求电池可拆卸性

Battery Reg Art.74
└─ 要求OEM提供拆解和回收信息（Battery Passport）

ADR危险品运输协议
└─ 电池作为Class 9 UN3480/3481危险品
```

BEV电池拆解信息是IDIS中最复杂、最关键的部分。不当拆解可能导致：
- 工人高压电击（致命）
- 电池热失控（火灾/爆炸）
- 环境污染（电解液泄漏）

## 必须提供的信息

### 1. 电池识别

| 信息 | 说明 |
|------|------|
| 电池型号 | OEM内部编号 + 化学类型（LFP/NMC/NCA等） |
| 电池供应商 | CATL/BYD/LG/Panasonic等 |
| Battery Passport ID | 按Battery Reg（2027起强制）|
| 化学成分类别 | 锂离子/钠离子/固态等 |
| 电池包级别容量（kWh） | 50-150 kWh典型 |
| 标称电压 | 400V / 800V |
| 总质量（kg） | 250-600 kg |
| 循环次数（设计） | 1,500-4,000次 |

### 2. 位置和结构

| 信息 | 细节 |
|------|------|
| **整车中位置** | 底盘下方（大多数）/前舱（个别）/车身一体化（CTB/CTC） |
| **电池包尺寸** | 长×宽×高，含图示 |
| **固定点** | 螺栓位置、数量、规格、扭矩、松开顺序 |
| **模组结构** | 几个模组、模组间隔、可否单独取出 |
| **电芯级别** | 电芯数量、形状（方形/圆柱/软包）、CTB/CTC是否可单独取电芯 |

### 3. 电气系统

| 组件 | 信息 |
|------|------|
| **高压连接器** | 位置、规格、断开工具和顺序 |
| **低压信号连接器** | BMS通讯、温度传感器 |
| **Service Disconnect** | 位置（通常在后排座椅下或后备箱）、操作步骤 |
| **主继电器** | 位置、强制断开方法（紧急情况） |
| **保险丝** | 主高压保险位置、规格 |
| **接地电阻检测** | 拆卸后验证无高压的测量点 |

### 4. 热管理系统

| 信息 | 说明 |
|------|------|
| **冷却液类型** | 乙二醇基 / 其他 |
| **冷却液容量** | 5-15 L典型 |
| **管路布局** | 冷板位置、水泵位置、管路图 |
| **断开点** | 快接接头位置、断开工具 |
| **泄漏风险** | 拆解前是否需排空 |
| **冷却液处理** | 按ELV预处理处理 |

### 5. 高压安全程序

关键！任何BEV拆解都必须遵循LOTO（Lock-Out, Tag-Out）：

```
Step 1: 识别
├─ 检查车辆为BEV（橙色高压线束可见）
├─ 检查电池健康状态（SOC、故障记录）
└─ 识别电池型号和化学类型

Step 2: 预放电（可选，但推荐）
├─ 车辆ON状态，空调/加热全开
├─ 放电至SOC <30%（降低风险）
└─ 仅在电池健康时进行

Step 3: 隔离
├─ 熄火、钥匙远离车辆
├─ 拔出12V辅助电池
└─ 车辆进入"service mode"

Step 4: 断开Service Disconnect
├─ 穿戴PPE（绝缘手套1000V级、面罩、绝缘鞋、棉质长袖）
├─ 工具：绝缘扳手
├─ 按OEM指定顺序旋转/拔出断开开关
└─ 锁定（Lock-out）防止意外重新连接

Step 5: 等待电容放电
├─ 等待5-15分钟（OEM规定）
├─ 此期间禁止触摸高压部件
└─ 观察指示灯/显示

Step 6: 验证无电压
├─ 使用CAT III 1000V万用表
├─ 测量指定测量点
├─ 确认< 60V DC（安全阈值）
└─ 验证后方可进行下一步

Step 7: 拆卸
├─ 按固定点顺序松开（对称松开避免应力）
├─ 使用吊具（电池包250-600 kg）
└─ 断开低压信号和冷却液
```

### 6. 拆卸工具要求

| 工具 | 规格 |
|------|------|
| 绝缘扳手 | 1000V耐压，CAT III |
| 绝缘手套 | Class 0或以上（1000V） |
| 绝缘鞋/垫 | EN 61111 |
| 面罩 | 弧闪防护 |
| 万用表 | CAT III 1000V，校准证书 |
| 吊具 | 承重≥1吨，电池连接点适配 |
| 拖车 | 防震，绝缘 |
| 灭火器 | Class D（金属/锂火）或水基 |
| 热成像仪 | 可选，监测电池温度 |

### 7. 应急处理

| 情况 | 应对 |
|------|------|
| **电池发热**（>60°C） | 立即停止作业，隔离，冷却（水或砂） |
| **冒烟** | 疏散20m，呼叫消防（注明锂电火灾），大量水淋 |
| **明火** | 不尝试扑灭，撤离50m，等待专业消防 |
| **电解液泄漏** | PPE+通风，吸收材料，按化学废物处理 |
| **机械损伤** | 不触碰破损处，按危险电池处理 |

### 8. 运输信息

| 信息 | 说明 |
|------|------|
| **ADR分类** | UN3480 锂离子电池（未装设备）或 UN3481（装于设备中） |
| **包装要求** | 防震/防穿刺/防接地外箱，内衬阻燃材料 |
| **SOC要求** | <30%运输（安全考虑） |
| **温度要求** | 运输温度通常10-25°C |
| **数量限制** | 单一包装SOC+质量限制 |
| **文件** | ADR运输文件、SDS、电池健康报告 |

### 9. 受损电池处理

按Battery Reg和ADR，受损电池分为：

| 级别 | 判定 | 处理 |
|------|------|------|
| 0级（完好） | 无损伤 | 正常回收流程 |
| 1级（轻微） | 外壳划伤但密封 | 加强包装运输 |
| 2级（中度） | 变形但未破裂 | 专用隔离容器 |
| 3级（重度） | 电解液泄漏/短路 | 热管理容器，专业运输 |
| 4级（失控） | 热失控发生过 | 灭火后冷却24h+，专用设施 |

## BEV vs HEV vs PHEV差异

| 类型 | 电池容量 | 拆解复杂度 | 风险 |
|------|---------|----------|------|
| **HEV（混动）** | 1-2 kWh | 中 | NiMH低风险，Li-ion中等 |
| **PHEV** | 8-20 kWh | 中-高 | 类似BEV但规模小 |
| **BEV** | 50-150 kWh | 高 | 最高风险 |

## CTB/CTC电池的特殊挑战

### Cell-to-Body (CTB) / Cell-to-Chassis (CTC)

将电池集成到车身/底盘的设计：

| 优势 | 拆解挑战 |
|------|---------|
| 车重减轻、空间利用率高 | 电池与车身无法分离 |
| 整车刚性提升 | 报废车辆整车需危险品处理 |
| 成本降低 | 材料回收困难 |

**法规冲突：** Battery Reg Art.11要求电池可拆卸——CTB/CTC设计可能与此冲突，IDIS需明确说明。

**主要采用OEM：**
- 比亚迪e Platform 3.0（CTB）
- 特斯拉 4680结构电池包
- 宁德时代麒麟电池（部分CTP）

## 第一响应者信息

车辆事故现场，消防/救援人员需要快速访问：

| 信息 | 访问方式 |
|------|---------|
| 电池位置 | 车身救援卡（Rescue Card）- ISO 17840 |
| 高压系统图 | 同上 |
| Service Disconnect位置 | 同上 |
| 电池化学类型 | 影响灭火策略 |
| 二维码/RFID | 车辆可扫描获取详情 |

### ISO 17840系列

| 标准 | 内容 |
|------|------|
| ISO 17840-1 | 救援卡（Rescue Sheet）格式 |
| ISO 17840-2 | 紧急响应信息结构 |
| ISO 17840-3 | 车辆上的紧急响应二维码 |
| ISO 17840-4 | 高压系统信息传达 |

## 合规检查清单

- [ ] IDIS中是否有完整BEV电池信息？
- [ ] 电池位置、尺寸、质量是否准确？
- [ ] Service Disconnect操作步骤是否详细？
- [ ] 等待电容放电时间是否明确？
- [ ] 验证无电压的测量点是否指定？
- [ ] 绝缘PPE要求是否列出？
- [ ] 起吊设备要求（≥1吨）是否说明？
- [ ] 冷却液处理是否纳入？
- [ ] 应急处理（热失控、火灾）是否包含？
- [ ] ADR运输要求是否说明？
- [ ] 受损电池分级处理是否描述？
- [ ] CTB/CTC特殊拆解是否说明？
- [ ] 第一响应者救援卡（ISO 17840）是否提供？
- [ ] 与Battery Passport是否协调？

## 常见不合规情况

1. **高压安全步骤简化**："断开高压插头后拆卸"一句话，缺失验证步骤。
2. **测量点未指定**：告诉拆解者"验证无电压"但未说在哪里测。
3. **PPE要求不详**：仅说"绝缘手套"未说Class。
4. **热失控应急缺失**：未说明冒烟/发热应如何应对。
5. **CTB/CTC拆解不可行**：声明"按常规拆除"但CTB实际不可独立取下。
6. **运输信息缺失**：未提ADR UN3480要求。
7. **受损电池无分级**：仅一种处理流程，未分级。
8. **救援卡缺失**：第一响应者（消防）无法快速获取关键信息。
9. **Battery Passport脱节**：IDIS数据与Battery Passport电池信息不一致。

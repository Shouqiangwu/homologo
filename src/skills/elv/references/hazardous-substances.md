# 有害物质限制和Annex II豁免

> ELV Directive 2000/53/EC Art.4(2) + Annex II

---

## 四种受限有害物质

| 物质 | 限值 | 适用范围 |
|------|------|---------|
| 铅(Pb) | 禁止使用（有Annex II豁免） | 所有车辆材料和零部件 |
| 汞(Hg) | 禁止使用（极少豁免） | 同上 |
| 镉(Cd) | 禁止使用（极少豁免） | 同上 |
| 六价铬 Cr(VI) | 禁止使用（有Annex II豁免） | 同上 |

**豁免清单：** 见 reach/references/exemptions.md 中的Annex II豁免表。

## 合规检查清单

- [ ] 车辆材料是否不含上述四种有害物质（或有有效豁免）？
- [ ] 豁免到期日是否已核查？
- [ ] 供应商是否提供了有害物质不使用声明？

# 超标罚款风险评估

> EU 2019/631 Art.8 + Annex III（Excess Emissions Premium）

---

## 罚款计算公式

```
超标罚款 = 超标克数 × €95 × EU新车注册量
```

| 变量 | 定义 |
|------|------|
| 超标克数 | 制造商车队平均CO2 − 制造商特定目标（g/km） |
| €95 | 固定单价（EU 2019/631 Art.8(1)） |
| 注册量 | 该日历年在EU注册的新车数量 |

## 真实案例与预测（2020-2025）

| 制造商（示例） | 年份 | 超标 | 估算罚款 |
|--------------|------|------|---------|
| Volkswagen Group | 2020 | ~0.5 g/km | ~€100M |
| PSA/Stellantis | 2020 | 达标 | €0 |
| Jaguar Land Rover | 2020 | ~8 g/km（豁免后仍有罚款） | €40M+ |
| BMW Group | 2021 | 达标（借助i系列ZEV） | €0 |
| 欧盟车企整体 | 2025预估 | 约€15B总罚款风险 | 基于-15%新目标 |

## 2025年冲击分析（新-15%目标）

| 制造商类型 | 2025合规难度 | 典型超标预期 |
|-----------|------------|------------|
| BEV纯电品牌（Tesla/蔚来/小鹏） | 无罚款风险 + 可出售pooling额度 | - |
| 混合大车企（VW/BMW/Stellantis） | 需大幅提升BEV占比 | 3-10 g/km |
| 小众豪华（Porsche/Ferrari等） | 可用小批量豁免 | 视豁免而定 |
| 传统ICE重点品牌（JLR/Mazda） | 高风险 | 5-15 g/km |
| 中国新品牌（比亚迪/奇瑞欧洲版） | 取决于产品组合 | 变数大 |

## 2025-2027缓冲期（提案）

欧委会2025年3月提出修正案，允许2025-2027年按**三年平均**合规计算：
- 2025超标但2026/2027下降，三年平均达标→免罚
- 该灵活性不适用于2028起

**对中国车企的影响：** 有BEV主力产品的中国品牌可利用2025较宽松窗口建立市场份额，无需急于降ICE销量。

## 罚款对冲策略

### 策略1：Pooling（加入排放池）

| 操作 | 收益 | 成本 |
|------|------|------|
| 加入Tesla/比亚迪等BEV品牌的pool | 利用其0g/km平均值摊薄超标 | 支付pool管理费（按EV车辆数结算，约€500-2000/辆） |
| 组建自己的pool | 内部车队统一管理 | 需跨品牌合规协调 |

### 策略2：增加ZLEV占比

2025-2029 ZLEV基准15%，超过基准每1个百分点可放宽1%目标（最多5%）：
- 若车队ZLEV达20%，目标从93.6放宽至98.3 g/km
- 等于每1个额外ZLEV车，间接抵消~3 g/km的ICE超标

### 策略3：延迟ICE新车型投放

| 场景 | 决策 |
|------|------|
| 2025-2027仍销ICE | 投放高销量低CO2配置 |
| 2028起 | 加速BEV/PHEV切换 |
| 2035起 | 仅ZEV |

### 策略4：Eco-innovation积分

按EU 2023/2534，制造商可申请使用"生态创新技术"获得CO2减免积分（单项最多7 g/km，但计算复杂）：
- 高效空调、太阳能车顶、LED大灯等
- 需ECHA认可并通过测试验证

## 风险量化模型

**简化公式：**
```
年度罚款风险(€) = max(0, 车队平均CO2 - 制造商特定目标) × 95 × 年注册量
```

**中国车企进入EU的典型场景测算：**

假设某中国品牌2026年EU销量：
- BEV 60,000辆（0 g/km）
- PHEV 15,000辆（45 g/km，2025新UF）
- ICE 25,000辆（130 g/km）

车队平均 = (60000×0 + 15000×45 + 25000×130) / 100000 = 39.25 g/km

若制造商特定目标为93.6 g/km → 无超标 → 罚款€0，且可卖pooling额度。

反之若仅销售ICE 100,000辆平均130 g/km → 超标36.4 g/km → **罚款€346M**（公司破产级风险）。

## 合规检查清单

- [ ] 是否计算了当前车队平均CO2？
- [ ] 是否了解制造商特定目标（Annex I Part A Point 6公式）？
- [ ] 是否评估了pooling的财务收益/成本？
- [ ] 是否监控了ZLEV占比与15%基准的差距？
- [ ] 是否考虑申请Eco-innovation积分？
- [ ] 是否有季度滚动的罚款风险预测？
- [ ] 财务部门是否预留了罚款准备金？

## 常见不合规情况

1. **低估注册量增长**：初期按1,000辆规划不报，后期超过1,000辆门槛未及时报告。
2. **未与pool签约就销售**：pooling必须在日历年前协议到位，事后无法追加。
3. **ZLEV资格理解错误**：将48V轻混误算为ZLEV（ZLEV定义<50g/km）。
4. **忽略小批量豁免到期**：10,000-300,000辆级的豁免2028年到期后未准备应对。

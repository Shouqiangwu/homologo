# 拆解信息手册（ELV Art.8）

> ELV Directive 2000/53/EC Art.8 + 新ELV提案要求

---

## 法规要求

ELV Art.8要求车辆制造商**在新车型投放市场后6个月内**，向授权处理设施（Authorised Treatment Facility, ATF）提供拆解信息。目的是促进高效的预处理、拆解和回收，确保达到Rcyc/Rcov目标。

## 信息内容要求

### 必须涵盖的内容

| 信息类别 | 具体要求 |
|---------|---------|
| **车辆识别** | VIN编码规则、车型识别、生产年份范围 |
| **预处理信息** | 所有液体排放点（燃油、机油、冷却液、制动液、转向液、空调制冷剂、窗洗液）|
| **安全气囊/ETS** | 爆炸装置的位置、中和方法、拆除步骤 |
| **起动电池** | 位置、拆除步骤（铅酸/锂电） |
| **催化转化器** | 位置、拆除步骤 |
| **空调系统** | 制冷剂类型（R-1234yf/R-744）、充注量、回收方法 |
| **高压电池**（EV） | 位置、质量、危险性等级、断电/拆卸步骤 |
| **高压电系统**（EV） | 线束路径、断电步骤、隔离措施 |
| **主要零部件位置** | 仪表板、座椅、车门、保险杠、动力总成位置 |
| **材料标识位置** | 塑料件上ISO 1043/11469标识的位置说明 |
| **有害物质位置** | 含Pb/Cr(VI)等有害物质的零部件位置（按Annex II豁免） |
| **重新使用信息** | 可直接重用的零部件清单和拆卸方法 |

### 推荐扩展内容

| 内容 | 原因 |
|------|------|
| 轮胎规格 | 便于回收企业按规格分类 |
| 玻璃位置和类型（钢化/夹层） | 不同回收路径 |
| 稀土磁体位置 | 电机、扬声器等应用 |
| 金属合金成分 | 铝合金/镁合金的具体型号 |

## 提交方式

### 通过IDIS平台（主流）

IDIS（International Dismantling Information System, www.idis2.com）是行业认可的拆解信息标准平台：

| 项目 | 说明 |
|------|------|
| 运营方 | IDIS Consortium（Stellantis/Ford/Toyota等OEM联合体） |
| 订阅 | OEM作为数据提供方，拆解企业作为使用方 |
| 数据格式 | 标准化XML schema + 图形/视频附件 |
| 语言 | 英文（强制）+ 可选其他EU语言 |
| 更新 | 每季度或按车型变更更新 |

### 其他可接受方式

- OEM自建拆解信息网站（需满足ATF免费获取要求）
- 纸质拆解手册（已不常见）

## 时间要求

| 节点 | 要求 |
|------|------|
| 上市后6个月内 | 完整拆解信息必须可获取 |
| 车型变更 | 影响拆解的改款（如电池规格变化）需更新信息 |
| 法规变更 | 如Annex II修订涉及新豁免标识，需更新 |
| 保存期 | 车型停产后至少15年（覆盖平均车辆生命周期） |

## 信息质量标准

### 准确性

- 拆卸步骤必须与实际车辆一致（非设计早期版本）
- 所有工具/扭矩/尺寸数据需准确
- 安全警告（高压电、气囊）必须明确

### 可用性

- 直观的图示/照片（不仅文字）
- 步骤编号和逻辑顺序
- 专业术语与通用术语并存（考虑各国拆解企业差异）

### 完整性

- 覆盖所有必填信息类别
- 不遗漏小量但关键的部件（如双离合器液压单元、电池管理系统BMS）

## EV特殊要求

### 高压安全（关键）

| 步骤 | 要求 |
|------|------|
| 1. 识别 | 如何识别EV（橙色高压线束、EV标识） |
| 2. 隔离 | 如何进入"维修模式"（Service Mode） |
| 3. 断电 | 拔出Service Disconnect的位置和步骤 |
| 4. 验证 | 如何测量确认无高压（万用表测量点） |
| 5. 等待 | 电容放电等待时间（通常≥5-10分钟） |
| 6. 锁定 | LOTO（锁定-挂牌）程序 |

### 电池包拆卸

| 信息 | 说明 |
|------|------|
| 电池包位置 | 底盘图示 |
| 电池包质量 | 用于选择起重设备 |
| 固定点 | 螺栓位置、扭矩、顺序 |
| 电气连接 | 高压连接器、信号连接器的断开方式 |
| 热管理连接 | 冷却液管路的断开和收集 |
| 拆卸工具 | 专用工具型号（避免普通工具损伤） |
| 运输要求 | ADR危险品分类、包装要求 |
| 应急处理 | 热失控初期迹象、隔离区域、冷却措施 |

## 新ELV提案的强化要求

| 现行要求 | 新ELV提案 |
|---------|----------|
| Art.8原则性要求 | 详细化、标准化 |
| 6个月时限 | 可能缩短至3个月 |
| 纸质/网页均可 | 强制数字化 + 材料护照协调 |
| OEM自主决定内容深度 | 标准化内容清单强制 |
| 无性能评估 | 增加拆解时间/效率评估指标 |

## 合规检查清单

- [ ] 是否通过IDIS或等效平台提供了拆解信息？
- [ ] 是否在新车型上市6个月内完成提交？
- [ ] 预处理信息（油液、气囊、电池、催化）是否完整？
- [ ] EV车型的高压安全步骤是否详细？
- [ ] 电池包拆卸和运输要求是否包含？
- [ ] 材料标识位置是否说明？
- [ ] 有害物质位置（Annex II豁免）是否标注？
- [ ] 信息是否提供英文版本？
- [ ] 车型改款时是否及时更新？
- [ ] 拆解企业能否免费获取信息？

## 常见不合规情况

1. **超6个月时限**：新车型EU上市后8-12个月才完成IDIS数据提交。
2. **EV电池信息简陋**：仅"断开高压插头后拆卸"一句话，未提供完整LOTO程序和应急处置。
3. **信息与实际车辆不一致**：拆解手册基于早期样车，量产车结构有差异。
4. **仅中文版本**：中国车企直接提交中文拆解手册，违反英文必备要求。
5. **有害物质位置遗漏**：含Pb焊料的ECU、含铅铜合金衬套等位置未标注。
6. **停产车型信息消失**：车型停产后OEM停止维护IDIS数据，违反15年保存期。
7. **拆解企业无法访问**：IDIS订阅费过高阻碍小拆解企业获取信息。
